Использование профилей

Если вы используете один и тот же компьютер для доступа в Интернет на работе и дома, для вас может оказаться удобным использование двух различных наборов закладок, учетных записей электронной почты, адресных книг и настроек &brandShortName;. Для членов одной семьи, использующих одну и ту же копию браузера, может быть полезной возможность иметь отдельный набор настроек для каждого из них.

Менеджер профилей позволяет создавать различные профили, с каждым из которых связан отдельный набор закладок, настроек, учетных записей и т.д. Профиль по умолчанию автоматически создается при установке браузера на компьютер. Если в дальнейшем вы создадите один или несколько дополнительных профилей, при каждом запуске программа предложит вам выбрать профиль для данного сеанса работы.

Создание нового профиля

Чтобы создать новый профиль:

  1. Откройте Менеджер профилей:
    • Если &brandShortName; запущен: Откройте меню "Инструменты" и выберите пункт "Выбрать другой профиль".
    • Если &brandShortName; закрыт: Откройте меню "Пуск" и последовательно выберите пункты "Программы" - "&brandShortName;" - "Profile Manager".Удерживайте клавишу Option при запуске &brandShortName; из Finder или дока.Наберите в командной строке: ./seamonkey -profilemanager
  2. В окне Менеджера профилей нажмите кнопку "Упорядочить профили", затем нажмите кнопку "Создать", просмотрите краткую информацию о профилях и нажмите кнопку "Далее".
  3. Введите имя для нового профиля. Оно может быть любым - при желании вы можете использовать ваше реальное имя или имя, характеризующее назначение профиля (например, "Работа").
  4. В диалоговом окне будет указан предложенный программой каталог для хранения закладок и другой информации, относящейся к данному профилю. Если вы хотите изменить его, нажмите кнопку "Выбрать папку".
  5. Если у вас установлено несколько языковых пакетов, нажмите кнопку "Выбрать язык". В открывшемся диалоговом окне выберите язык для данного профиля.
  6. Нажмите кнопку "Готово".

Удаление или переименование профиля

Если вы не используете созданный профиль, вы можете удалить его. Чтобы удалить или переименовать профиль:

  1. Откройте Менеджер профилей:
    • Если &brandShortName; запущен: Откройте меню "Инструменты" и выберите пункт "Выбрать другой профиль". Замечание: Вы не можете удалить профиль, который используется в данный момент.
    • Если &brandShortName; закрыт: Откройте меню "Пуск" и последовательно выберите пункты "Программы" - "&brandShortName;" - "Profile Manager". Удерживайте клавишу Option при запуске &brandShortName; из Finder или дока. Наберите в командной строке: ./seamonkey -profilemanager
  2. В окне Менеджера профилей нажмите кнопку "Упорядочить профили". Чтобы удалить профиль, выберите его из списка и нажмите кнопку "Удалить". В окне подтверждения удаления выберите один из возможных вариантов:
    • Удалить файлы: Удаляет профиль целиком со всеми хранимыми там данными (закладками, настройками, почтовыми ящиками и т.д.). При выборе этого варианта убедитесь, что вам не нужна никакая информация из этого профиля.
    • Не удалять файлы: Удаляет профиль из списка доступных профилей, но сохраняет папку профиля. При выборе этого варианта данные хранящиеся в этом профиле не будут удалены.
    Чтобы переименовать профиль, выберите его из списка, нажмите кнопку "Переименовать" и следуйте дальнейшим указаниям.

Перемещаемые профили

Перемещаемые профили позволяют хранить ваш профиль и его настройки на центральном сервере или в общей папке. Преимущество перемещаемого профиля заключается в том, что ваши настройки будут одинаковы, даже если вы работаете на разных машинах. Например, возможно вы захотите использовать &brandShortName; на работе и дома, и при этом иметь одинаковый набор закладок, адресную книгу и другие настройки, или например ваша компания хочет, чтобы её работники имели возможность использовать свой профиль &brandShortName; на любом компьютере.

Перемещаемые профили могут хранится на HTTP или FTP сервере. Вам понадобится имя пользователя и пароль для входа на этот сервер, чтобы вы имели права на запись в соответствующий каталог сервера для выгрузки и загрузки вашего профиля. В качестве альтернативы, для компьютеров использующих общие папки, вы можете сохранять профиль в локальную папку используя метод копирования.

При каждом запуске &brandShortName;, ваш профиль загружается и при каждом завершении работы &brandShortName;, ваш профиль выгружается.

Настройки перемещаемого профиля

В этом разделе описано, как использовать панель настроек Перемещаемого профиля. Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню &brandShortName;"Правка" и выберите пункт "Настройки".
  2. Выберите категорию "Перемещаемый профиль".
  3. Установите опцию Включить хранение профиля на сервере для активации хранения текущего профиля на сервере.
  4. Выберите из выпадающего списка одну из следующих опций:
    • HTTP / FTP: Выберите эту опцию, если хотите, чтобы информация профиля копировалась на FTP или HTTP сервер.
      • Основной URL: Введите URL каталога, где будет хранится перемещаемый профиль. Это может быть FTP, HTTP или HTTPS URL. FTP возможно является самый легким в настройке протоколом для домашней сети. В других случаях, особенно в случае передачи информации через Интернет, настоятельно рекомендуется использовать HTTPS, иначе, все ваши сохраненные пароли и закладки (в зависимости от выбора компонентов) могут быть прочтены лицами, находящимися в той же сети.

        Примеры:
        <https://user@http-server/dir/>
        <ftp://user@ftp-server//home/user/dir/>
        <ftp://user@ftp-server/dir/>

      • Имя пользователя: Введите имя пользователя, которое будет использоваться для доступа к FTP или HTTP серверу.
      • Сохранить пароль: Установите флажок, если хотите сохранить пароль для введённого вами имени пользователя.
      • Пароль: Наберите пароль для введённого вами имени пользователя.
    • Копирование файла: Выберите эту опцию, если хотите, чтобы информация профиля копировалась в локальную папку.
      • Нажмите кнопку Обзор и выберите каталог, в который вы хотите копировать и из которого вы хотите получать информацию профиля.

Настройки перемещаемого профиля - Выбор компонентов

По умолчанию, только закладки, cookies, и личная адресная книга хранится на сервере в перемещаемом профиле. Если все компоненты выделены серым, убедитесь, что использование Перемещаемых профилей включено. Чтобы включить или отключить перемещение другой информации профиля, выполните следующие действия:

  1. Откройте меню &brandShortName;"Правка" и выберите пункт "Настройки"
  2. Выберите категорию "Перемещаемый профиль" и подкатегорию "Выбор компонентов". При необходимости щелкните по небольшому значку слева от категории "Перемещаемый профиль", чтобы развернуть список подкатегорий.
  3. Выберите компоненты, которые вы хотите включить в процесс синхронизации при использовании перемещаемого профиля.

Вы не можете перемещать в профиль на сервере локальную почту, частично из-за того, что это может занять слишком много времени. Используйте для этого протокол IMAP.

Примечание: Если вы находитесь на медленном соединении, не забывайте, что некоторые файлы могут быть довольно большого размера, так что использование перемещаемого профиля может существенно повлиять на скорость запуска программы. &brandShortName; должен будет загрузить на локальную машину все выбранные вами файлы перед тем, как начать работу и сохранить их на удаленный сервер при выходе из &brandShortName;.

Для получения более подробной информации о настройке Перемещаемого доступа на сервере, обратитесь к онлайновому документу mod_roaming - Roaming Access module for Apache или Manually Implementing Roaming Access.
