Использование клавиатурных сокращений

Для описания клавиатурных сокращений в этом документе используется следующий формат:

Пример:

Команда Windows Mac OS Linux или Unix
Копировать Ctrl+C Cmd+C Ctrl+C

Для выполнения команды необходимо нажать указанные клавиши одновременно. Например, чтобы выполнить команду "Копировать" в Windows, нужно одновременно нажать клавиши Ctrl (Control) и C.

Совет: знак плюса (+) означает, что клавиши, объединяемые им, нужно нажать одновременно. Если в комбинации несколько знаков "плюс" (например, Ctrl+Shift+C), это значит, что все три клавиши следует нажать одновременно.

Сокращения:

  • Ctrl = клавиша Control
  • Cmd = клавиша Command на клавиатуре Macintosh

Примечание: некоторые комбинации выполняют различные функции в зависимости от расположения курсора (фокуса). Например, если вы в Windows нажмете кнопку Home во время просмотра веб-страницы, &brandShortName; прокрутит документ на начало. Но если вы нажмете эту же клавишу в Windows, когда курсор будет находится в поле ввода текста, это приведет к перемещению курсора в начало поля ввода.
