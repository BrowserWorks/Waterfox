    • Прокручивать документ на: Прокручивать страницу на число строк (или символов для горизонтальной прокрутки), указанное в поле ввода.
      • Системные настройки: Установите этот флажок, чтобы прокручивать страницу на число строк (символов), указанное в системных настройках.
    • Прокручивать на страницу вверх(влево) или вниз(вправо): Прокручивать документ на целую страницу вверх или вниз. Эта функция предоставляет возможность более быстрого, но менее точного перемещения по документу.
    • Перемещаться вперед или назад в журнале посещений: Перемещаться вперед или назад на ранее просмотренные страницы.
    • Увеличивать или уменьшать текст: Увеличивать или уменьшать размер текста на веб-странице. Эта функция позволяет легко подобрать удобный размер шрифта или заставить текст помещаться в окне.

    Примечание: С каждой служебной клавишей может быть связана отдельная функция.

  • Если в вашей мыши отсутствует режим горизонтальной прокрутки, любые настройки в нижней панели Горизонтальная прокрутка будут проигнорированы.

Дополнительные настройки - Инспектор DOM

Инспектор DOM это необязательный компонент предназначенный для разработки веб-приложений. В этом разделе описано как использовать его панель настроек. Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню &brandShortName;"Правка" и выберите пункт "Настройки"
  2. В категории "Дополнительно" выберите подкатегорию "Инспектор DOM". (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

При выборе элемента, Инспектор DOM может автоматически подсвечивать его путем мигания его или его границы:

  • Подсвечивать выбранный элемент: Отметьте эту опцию, чтобы включить подсвечивание элемента который вы выбираете. По умолчанию вокруг элемента появляется мигающая граница.
    • Цвет границы: Выберите цвет границы вокруг элемента.
    • Ширина границы: Введите ширину границы вокруг элемента.
    • Длительность подсветки: Введите длительность времени (в миллисекундах) в течении которой будет производиться мигание.
    • Скорость мигания при подсветке: Введите временной интервал (в миллисекундах) между миганиями.
    • Обратить цвета: Отметьте эту опцию, чтобы обратить цвет выделенного элемента. Это вызовет мигание всего элемента, включая границу.

Дополнительные параметры - Система

В этом разделе описаны дополнительные настройки &brandShortName;, относящиеся к подкатегории "Система". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "Правка" и выберите пункт "Настройки".
  2. В категории "Дополнительно" выберите подкатегорию "Система" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете задать, для отображения каких файлов и поддержки каких протоколов операционная система должна использовать &brandShortName;.

  • Windows следует использовать &brandShortName; для открытия файлов следующих типов: Установите флажки для тех типов файлов, которые по умолчанию должны отображаться при помощи &brandShortName;.
  • Windows следует использовать &brandShortName; для поддержки следующих протоколов: Установите флажки для тех протоколов, которые должны поддерживаться при помощи &brandShortName;.
  • Предупредить меня, если другие приложения изменят эти параметры: Установите этот флажок, чтобы получать предупреждение в том случае, когда какое-либо приложение изменит параметры открытия файлов или поддержки протоколов, установленные в настройках &brandShortName;.

Совет: Чтобы сделать &brandShortName; браузером по умолчанию обратитесь к разделу Настройки Навигатора - Навигатор. Также установите флажок Предупредить меня, если другие приложения изменят эти параметры чтобы получать уведомления при попытках других программ стать программами по умолчанию вместо &brandShortName;.
