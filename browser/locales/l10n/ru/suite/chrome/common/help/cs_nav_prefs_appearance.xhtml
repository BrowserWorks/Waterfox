Настройки внешнего вида - Внешний вид

В этом разделе описаны общие настройки внешнего вида &brandShortName;. Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. Выберите категорию "Внешний вид"

В панели настроек внешнего вида вы можете выбрать какие компоненты будут использованы при запуске &brandShortName;.

  • При запуске &brandShortName; открывать: Установите флажки для тех компонентов, которые должны открываться при запуске &brandShortName;.
  • Показывать панели инструментов как:
    • Значки и текст: Выберите этот переключатель, чтобы на панели инструментов были показаны значки с текстовым описанием под каждым из них..
    • Значки: Выберите этот переключатель, чтобы на панели инструментов были показаны только значки.
    • Текст: Выберите этот переключатель, чтобы на панели инструментов были показаны только текстовое описание значков.
  • Показывать всплывающие подсказки: Установите этот флажок, если вы хотите, чтобы при наведении курсора мыши на элементы интерфейса &brandShortName; и некоторые сайты появлялась всплывающая подсказка.
  • Показывать значки сайтов: Установите этот флажок, если вы хотите, чтобы вместо значка закладки отображался значок текущего сайта. Значки сайтов отображаются слева на панели адреса и вкладках.
  • Использовать плавную прокрутку: Установите этот флажок чтобы включить плавную прокрутку. Если этот параметр включен и вы нажали клавишу Page Down, то вместо резкого прыжка вниз документ будет плавно прокручен на страницу вниз.
  • Использовать автоматическое изменение размеров изображений: Установите этот флажок если вы хотите чтобы &brandShortName; автоматически уменьшала большие изображения, открытые в отдельном окне/вкладке, таким образом, чтобы они помещались в окне/вкладке целиком. Для показа изображения в полную величину щелкните по нему мышью.

Настройки внешнего вида - Шрифты

В этом разделе описаны настройки внешнего вида &brandShortName;, относящиеся к подкатегории "Шрифты". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. В категории "Внешний вид" выберите подкатегорию "Шрифты" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете задать шрифты для отображения веб-страниц.

Примечание: Для некоторых наборов символов отдельные стили шрифтов могут быть недоступны. Это означает, что в операционной системе не установлены шрифты данного стиля с необходимыми символами.

  • Шрифты для: Выберите набор символов, для которого вы хотите задать шрифты. Например, чтобы установить шрифты по умолчанию для страниц на русском и других языках, использующих кириллицу, выберите пункт Кириллица. Для шрифта/языка не находящегося в списке, выберите Другие языки..
    • Пропорциональный: Выберите, шрифт какого типа должен использоваться в качестве пропорционального шрифта - с засечками (например, Times Roman) или без засечек (например, Arial). Вы также можете установить размер пропорционального шрифта. В пропорциональном шрифте символы имеют различную ширину в зависимости от их рисунка.
    • С засечками: Выберите гарнитуру, которая должна использоваться в качестве шрифта с засечками при отображении веб-страниц.
    • Без засечек: Выберите гарнитуру, которая должна использоваться в качестве шрифта без засечек при отображении веб-страниц.
    • Курсив: Выберите гарнитуру, которая должна использоваться в качестве курсива (шрифта, стилизованного под рукописный) при отображении веб-страниц.
    • Декоративный: Выберите гарнитуру, которая должна использоваться в качестве декоративного шрифта при отображении веб-страниц.
    • Моноширинный: Выберите гарнитуру, которая должна использоваться в качестве моноширинного шрифта (например, Courier). В моноширинном шрифте все символы имеют одинаковую ширину.
    • Наименьший размер шрифта: Выберите наименьший размер шрифта который может быть использован при отображении веб-страниц.
  • Разрешить документам использовать другие шрифты: Установите этот флажок если хотите разрешить веб-страницам использовать гарнитуры и размеры шрифтов, заданные их авторами.

Настройки внешнего вида - Цвета

В этом разделе описаны настройки внешнего вида &brandShortName;, относящиеся к подкатегории "Цвета". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. В категории "Внешний вид" выберите подкатегорию "Цвета" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете задать цвета для отображения текста и фона веб-страниц.

  • Текст и фон: Щелкните по одному из цветных прямоугольников, чтобы выбрать цвет для отображения текста или фона веб-страниц.
    • Использовать системные цвета: Установите этот флажок, чтобы для текста и фона использовались цвета, установленные для них в системе.
  • Цвет ссылок: Щелкните по одному из цветных прямоугольников, чтобы выбрать цвет для отображения просмотренных, активных и непросмотренных ссылок на веб-страницах.
    • Подчеркивать ссылки: Установите этот флажок, если вы хотите, чтобы ссылки отображались подчеркнутым текстом.
  • Если веб-страница предоставляет свои цвета и фон:
    • Всегда использовать цвета и фон, указанные веб-страницей: При отображении веб-страниц использовать цвета и фон, заданные автором страницы.
    • Использовать выбранные мной цвета, игнорируя указанные цвета и фон: При отображении веб-страниц использовать цвета и фон, указанные в настройках &brandShortName;, игнорируя цвета и фон, заданные автором страницы.

Настройки внешнего вида - Темы

В этом разделе описаны настройки внешнего вида &brandShortName;, относящиеся к подкатегории "Темы". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. В категории "Внешний вид" выберите подкатегорию "Темы" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете изменять общий вид и стиль интерфейса &brandShortName;, выбирая различные темы.

  • Установленные темы: Чтобы изменить общий вид (стиль интерфейса) &brandShortName;, выберите в списке одну из тем.
    • Удалить тему: Нажмите эту кнопку, чтобы удалить выбранную тему.
  • Новые темы: Нажмите эту ссылку чтобы перейти на веб-страницу, где вы можете загрузить другие темы для &brandShortName;

    Примечания:

    • Для переключения интерфейса на новую тему вам необходимо перезапустить &brandShortName;.
    • Тема(ы) по умолчанию не могут быть удалены.
    • Удалив тему, вы не сможете переключиться на нее в дальнейшем, не установив ее повторно.

Настройки внешнего вида - Языки

В этом разделе описаны настройки внешнего вида &brandShortName;, относящиеся к подкатегории "Языки". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".

В этой подкатегории вы можете выбрать текущий пакет локализации (язык интерфейса и справочной системы) для &brandShortName;.

  • Доступные языки: Выберите язык интерфейса и справочной системы &brandShortName;.
  • Установленные пакеты: Выберите пакет региональных установок для &brandShortName;. Текущий пакет региональных установок определяет информационное наполнение ряда элементов &brandShortName;, в частности, доступные разделы Боковой панели и набор закладок.
