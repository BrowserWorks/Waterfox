Настройки Навигатора - Навигатор

В этом разделе рассказано, как установить общие параметры Навигатора при помощи диалогового окна "Настройки". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. Выберите категорию "Навигатор".

Панель настроек навигатора позволяет вам настроить определенные настройки, относящиеся к запуску Навигатора, а также определяющие, какие кнопки видны на панелях инструментов.

  • Открывать при: Используйте выпадающий список совместно со расположенной ниже группой переключателей, чтобы определить какая страница будет отображена при запуске браузера или при открытии нового окна или вкладки:
      • Выберите Запуске Навигатора чтобы установить какая страница будет открыта при запуске Навигатора.
      • Выберите Создании окна чтобы установить какая страница будет открыта при создании нового окна Навигатора.
      • Выберите Создании вкладки чтобы установить какая страница будет открыта при создании новой вкладки Навигатора.
    2. Выберите один из следующих переключателей:
      • Пустую страницу: Навигатор отображает пустую страницу.
      • Начальную страницу: Навигатор загружает страницу, указанную в его настройках как начальная (указанную ниже в том же разделе настроек).
      • Последнюю просмотренную страницу: Навигатор загружает последнюю страницу, просмотренную в ходе предыдущего сеанса работы перед выходом из &brandShortName;.
  • Браузер по умолчанию: Позволяет установить &brandShortName; как браузер по умолчанию или показывает что он уже им является.
    • По умолчанию: Если эта надпись на этой кнопке не серого цвета, то, нажав на нее, вы установите &brandShortName; как ваш браузер по умолчанию.
  • Начальная страница: Введите в это поле адрес (URL) страницы, которую вы хотите использовать как начальную или сделайте одно из следующих действий:
    • Текущая страница: Нажатие этой кнопки установит страницу, отображаемую Навигатором в настоящий момент, в качестве начальной.
    • Текущая группа страниц: Если у вас открыто 2 или более вкладки Навигатора, нажатие этой кнопки установит их как Начальную группу страниц (группу вкладок открывающуюся в качестве начальной страницы). После нажатия этой кнопки в поле Адрес появится сообщение Начальная группа задана.

      Внимание: Если вы попытаетесь отредактировать поле Адрес после нажатия кнопки Текущая группа страниц, то ваша Начальная группа страниц будет утеряна.

    • Восстановить: Нажатие этой кнопки возвращает начальной странице значение по умолчанию.
    • Выбрать файл: Нажатие этой кнопки позволит выбрать нужный файл в диалоговом окне, если вы хотите использовать в качестве начальной страницу, сохраненную на локальном диске.
  • Выберите кнопки, которые вы хотите видеть на панелях инструментов: Установите флажки для тех кнопок, которые должны быть доступны в панелях инструментов.

    Кнопки "Перейти", "Поиск" и "Печать" расположены в правой части навигационной панели. Все остальные кнопки отображаются в личной панели. О том, как добавить к личной панели ваши собственные закладки, рассказано в разделе Личная панель.

Настройки Навигатора - Приложения

В этом разделе описаны настройки Навигатора, относящиеся к подкатегории "Вспомогательные приложения". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. В категории "Навигатор" выберите подкатегорию "Приложения" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете задать вспомогательные приложения, которые используются для отображения различных типов файлов в &brandShortName;.

  • Укажите, как должны обрабатываться файлы этих типов:
    • Типы файлов: Список типов файлов, которые использует &brandShortName;. Для выбранного типа отображается следующая информация:
      • Описание: Описание выбранного типа файлов.
      • Расширение: Расширение данного типа файлов.
      • Приложение: Описывает какое действие &brandShortName; должен предпринять при обработке файлов данного типа.
    • Новый тип: Нажмите эту кнопку, чтобы добавить новый тип файлов. Вы увидите следующее окно:
      • MIME-тип: Наберите тип MIME добавляемого вами типа файлов.
      • Описание: Наберите описание добавляемого вами типа файлов.
      • Расширение: Наберите расширение добавляемого вами типа файлов.
      • При необходимости обработки файла этого типа: Выберите одну из следующих опций:
        • Открывать с помощью приложения по умолчанию: Файл будет открыт с приложения в системе по умолчанию.
        • Открывать с помощью: Файл будет открыт с помощью приложения которое вы укажете. Для выбора нужного приложения используйте кнопку "Выбрать".
        • Сохранять на диск: Файл будет сохранен на диск.
      • Спрашивать перед открытием файлов этого типа: Выберите эту опцию чтобы при обработке файлов этого типа всегда появлялся диалог загрузки.
    • Изменить: Изменить параметры, заданные для данного типа файлов.
    • Удалить: Удалить выбранный тип файла и параметры, относящиеся к нему.
  • Служба поиска модулей:
    • Всегда использовать службу поиска модулей Netscape (PFS) для их загрузки: Выберите эту опцию чтобы всегда использовать службу поиска модулей Netscape (PFS) когда &brandShortName; потребуется новый модуль. Если эта опция отключена, то PFS будет использоваться только в том случае если расположение модуля не будет указано страницей которая его требует.

Если страница была открыта другим способом (например, путем перехода по ссылке), ее адрес не будет включаться в раскрывающийся список.

    • щелчке с удержанием Cmd или Cmd+Returnнажатии средней кнопки мыши, нажатии левой кнопки мыши с удержанием Ctrl или нажатии Ctrl+Enter на ссылках веб-страниц: Установите этот флажок, чтобы Навигатор автоматически загружал новую страницу в новой вкладке если вы щелкаете по ссылке удерживая клавишуCommand или удерживая клавишу Command нажимаете ссылку клавишей Returnсредней кнопкой мыши, левой кнопки мыши с удержанием Ctrl и нажимаете на ссылку используя Ctrl+Enter

      Совет: Используйте Поиск ссылок по мере набора, чтобы перейти к ссылке которую вы хотите открыть при помощи горячих клавиш на клавиатуре.

    • Нажатии CmdCtrl+ReturnEnter в панели адреса: Установите этот флажок, чтобы Навигатор автоматически загружал новую страницу в новой вкладке, если вы вводите адрес (URL) в панели адреса и нажимаете CmdCtrl+ReturnEnter.

    • Новой вкладке текущего окна: Открыть ссылку в новой вкладке вместо нового окна.
    • Новом окне: Открыть ссылку в новом окне. (Это значение по умолчанию и не изменяет поведение заданное веб-мастером страницы).
  • Ссылки из других приложений: Если &brandShortName; вызывается из другого приложения с адресом веб-страницы в качестве аргумента (как например щелчок по ссылке во внешней почтовой программе), вы можете определить как будет загружена вызываемая страница:
    • Текущей вкладке/окне: Открыть ссылку в текущей вкладке активного окна.
    • Новой вкладке текущего окна: Открыть ссылку в новой вкладке вместо нового окна.
    • Новом окне: Открыть ссылку в новом окне. (Это значение по умолчанию).

Чтобы убедиться, что &brandShortName; откроет новое окно, установите опцию Навигатор в разделе Настройки внешнего вида - Внешний вид.

Если вы хотите, чтобы &brandShortName; открывал новую вкладку вместо нового окна при его запуске и когда он уже запущен, удостоверьтесь что:

  • При запуске &brandShortName; из командной строки не используются ключи командной строки открывающие окна.
  • Сняты галочки со всех опций в разделе Настройки внешнего вида - Внешний вид.
  • Опция Ссылки из других приложений установлена в положение Новой вкладке текущего окна.

Настройки Навигатора - Загрузки

В этом разделе описаны настройки Навигатора, относящиеся к подкатегории "Загрузки". Если в данный момент вы не просматриваете эти настройки, выполните следующие действия, чтобы получить доступ к ним:

  1. Откройте меню "&brandShortName;Правка" и выберите пункт "Настройки".
  2. В категории "Навигатор" выберите подкатегорию "Загрузки" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).

В этой подкатегории вы можете задать параметры, относящиеся к загрузке файлов с веб-страниц с помощью менеджера Загрузок &brandShortName;.

  • При начале загрузки:
    • Открывать менеджер загрузок: Выберите этот вариант, чтобы при начале загрузки файла открывался Менеджер Загрузок который отображает состояние текущих и ранее произведенных загрузок. Состояние всех загрузок отражается в едином окне Менеджера Загрузок.
    • Открывать диалоговое окно состояния загрузки: Выберите этот вариант, чтобы при начале загрузки файла открывалось диалоговое окно состояния загрузки отражающее её статус. Состояние каждой загрузки отражается в отдельном окне.
    • Не открывать ничего: Выберите этот вариант чтобы загружать файлы не открывая никаких окон. При загрузке файлов состояние загрузок отображено не будет.
  • При сохранении файла:
    • Спрашивать куда сохранять файл, открывая по умолчанию: Выберите этот вариант, если вы всегда хотите выбирать папку, в которую будет сохранен файл. Выберите одну из следующих папок в качестве папки по умолчанию предлагаемой для сохранения файла:
      • Папку в которую был сохранен последний загруженный файл: Папкой по умолчанию будет папка в которую был сохранен последний загруженный файл.
      • Указанную папку загрузок: Папкой по умолчанию будет ваша Текущая папка для загрузок.
    • Автоматически сохранять файлы в указанную папку загрузок: Выберите этот вариант, если вы хотите сохранять файлы в вашу Текущую папку для загрузок и не хотите, чтобы &brandShortName; выдавал запрос на выбор папки для загрузки.
  • Текущая папка для загрузок: Используйте кнопку Выбрать папку для выбора папки, которую вы хотите использовать для сохранения загруженных файлов.
  • По завершении загрузки:
    • Подать звуковой сигнал: Установите флажок, если хотите, чтобы &brandShortName; подавал звуковой сигнал (обычно в формате WAV) по завершении загрузки файла. Используйте кнопку Обзор для выбора звукового файла. Чтобы прослушать выбранный вами звук, нажмите кнопку Просмотр.
    • Показать уведомление: Установите флажок, если хотите, чтобы &brandShortName; показывал уведомление на экране по завершении загрузки файла.

Дополнительная информация о работе загрузке файлов с веб-страниц приведена в разделе Плагины и загрузки.
