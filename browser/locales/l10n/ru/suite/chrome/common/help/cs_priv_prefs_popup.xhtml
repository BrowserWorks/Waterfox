Контроль за всплывающими окнами

Что такое всплывающие окна?

Всплывающие окна - это окна появляющиеся автоматически без вашего разрешения. Они бывают разного размера, но обычно не достигают размеров всего экрана. Некоторые всплывающие окна появляются над текущим окном браузера, некоторые под текущим окном браузера.

&brandShortName; позволяет вам установить контроль за всплывающими окнами с помощью Панели настроек всплывающих окон. Так как блокировка всплывающих окон отключена по умолчанию, вы должны включить её для предотвращения появления всплывающих окон в браузере.

&brandShortName; можно настроить так, чтобы при блокировке всплывающего окна происходило звуковое уведомление и в строке статуса отображался значок. Этот значок может быть использован чтобы добавить сайт, открывающий всплывающие окна, в список исключений, чтобы он мог открывать всплывающие окна снова.

Блокировка всплывающих окон может привести затруднениям при работе с некоторыми сайтами: Некоторые сайты, в том числе некоторые банковские сайты, используют всплывающие окна для выполнения важных функций. Блокировка всплывающих окон делает невозможным работу с данными функциями. Чтобы разрешить этим сайтам открывать всплывающие окна, заблокировав при этом всплывающие окна с других сайтов, вы можете добавить их в список разрешенных сайтов. Для получения более подробной информации обратитесь к разделу Приватность и защита - Всплывающие окна.

Блокирование всплывающих окон не всегда срабатывает: Хотя &brandShortName; блокирует большую часть всплывающих окон, некоторые сайты могут использовать другие методы открытия всплывающих окон, даже при включенной блокировке.

Разрешение открытия всплывающих окон с определенных веб-сайтов: После того как вы включили блокирование всплывающих окон, вы можете разрешить определенным веб-сайтам открывать всплывающие окна. Перейдите на нужный сайт, затем из меню "Инструменты" выберите подменю "Менеджер всплывающих окон", а в нем выберите пункт "Разрешить всплывающие окна этого сайта".

В следующем разделе описано как осуществлять контроль над всплывающими окнами через настройки и посредством значка контроля над всплывающими окнами.

Приватность и защита - Всплывающие окна

В этом разделе описано как использовать панель настроек всплывающих окон. Если в данный момент вы не просматриваете её, выполните следующие действия, чтобы получить доступ к ней:

  1. Откройте меню &brandShortName;"Правка" и выберите пункт "Настройки"
  2. В категории "Приватность и защита" выберите подкатегорию "Всплывающие окна" (при необходимости используйте значок слева от названия категории, чтобы развернуть список подкатегорий).
  • Блокировать всплывающие окна: Выберите эту опцию чтобы блокировать открытие всплывающих окон в Навигаторе.
  • Разрешенные сайты…: Нажмите эту кнопку, чтобы просмотреть и отредактировать список веб-сайтов которым разрешено открывать всплывающие окна.
    • Сайты со всплывающими окнами: Список веб-сайтов появляется при нажатии кнопки Разрешенные сайты…. В нем вы можете добавить или удалить веб-сайты которым разрешено открывать всплывающие окна.
    • Добавить: Нажмите эту кнопку, после того как вы набрали URL сайта, чтобы добавить его в список сайтов.
    • Удалить: Нажмите эту кнопку, чтобы удалить выбранный сайт из списка.
    • Удалить все: Нажмите эту кнопку, чтобы удалить все сайты из списка.

Если всплывающее окно было заблокировано

  • Проигрывать звук: Если вы хотите чтобы при каждом блокировании всплывающего окна Навигатор проигрывал звук, выберите эту опцию и введите в поле расположение звукового файла.
    • Выбрать: Нажмите эту кнопку, чтобы выбрать звуковой файл из диалогового окна вместо того чтобы набирать путь к нему вручную.
    • Просмотр: Нажмите эту кнопку, чтобы прослушать выбранный звуковой файл.
  • Показывать значок в строке состояния Навигатора: Выберите эту опцию, чтобы отобразить значок в панели статуса Навигатора при блокировании всплывающего окна. Примечание: После появления этого значка, он остается видимым до тех пор пока вы не посетите другой веб-сайт.

Использование значка уведомления о блокировании всплывающего окна для добавления сайта в список разрешенных: Вы можете использовать значок уведомления о блокировании всплывающего окна чтобы быстро добавить сайт в список сайтов которым разрешено открывать всплывающие окна. Щелкните по значку чтобы открыть список сайтов которым разрешено открывать всплывающие окна. При этом в поле адреса будет стоять адрес текущего сайта. Нажмите Добавить и OK для подтверждения вашего выбора.

Примечание:Блокирование всплывающих окон не всегда срабатывает и может препятствовать взаимодействию с некоторыми сайтами. Более подробную информацию о блокировании всплывающих окон можно найти в разделе Контроль за всплывающими окнами.
