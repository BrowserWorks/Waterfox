Инструменты разработки веб-приложений

В составе &brandShortName; имеется несколько инструментов имеющих своей целью помочь веб-разработчикам разрабатывать и отлаживать веб-приложения. Некоторые из них устанавливаются только при выборочной установке &brandShortName;. Для получения доступа к этим инструментам откройте меню "Инструменты" и выберите подменю "Веб-разработка".

  • Консоль JavaScript: Консоль в которой отображаются ошибки, предупреждения и уведомления при выполнении сценариев JavaScript в веб-приложениях и в самой &brandShortName;.
  • Инспектор DOM: Инструмент позволяющий исследовать и редактировать DOM любого веб-документа или приложения XUL.

    Ресурсы в Интернете (на английском языке):

    • Introduction to the DOM Inspector (Ian Oeschger)
    • DOM Inspector FAQ (mozilla.org)
    • Creating Applications with Mozilla - Appendix B3: The DOM Inspector
  • Отладчик JavaScript: также известный как Venkman, эта утилита позволяет производить отладку JavaScript в &brandShortName;. В ней имеется пошаговая отладка, поддержка точек останова и многие другие возможности.

    Ресурсы в Интернете (на английском языке):

    • Venkman JavaScript Debugger Homepage
    • Learning the JavaScript debugger Venkman (Svend Tofte)
    • Debugging JavaScript Using Venkman (webreference.com, article series)
    • Venkman Development Homepage
