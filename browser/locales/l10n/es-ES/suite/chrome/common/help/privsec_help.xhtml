Rastreo de usuarios

Estas opciones le permiten comunicar sus preferencias de rastreo a los sitios web y para bloquear activamente las solicitudes de rastreo de sitios conocidos por rastrear. El rastreo se refiere a que los sitios web (incluyendo anunciantes, proveedores de analíticas y sitios sociales) recogen y analizan datos comprensivos sobre sus patrones de navegación web, afectando por tanto a su privacidad. Para más información sobre este tema, vea ¿Por qué y cómo me rastrean los sitios web? En esta sección están disponibles las siguientes opciones:

  • Indicar a los sitios web que no quiero ser rastreado: marque esta casilla para indicar a &brandShortName; que informe a cada sitio web que visite explícitamente (junto con cualquier proveedor de contenido de terceros incrustado en sus páginas) que no quiere que se rastree sus hábitos de navegación. Si está marcada, se envía una solicitud de No rastreo a cada sitio web visitado.
  • Impedir actividades de rastreo de sitios conocidos: marque esta casilla para indicar a &brandShortName; que bloquee activamente las conexiones a sitios web que son conocidos por proporcionar servicios de rastreo. Puede que no vea ningún contenido procedente de estos sitios en la página web visitada.
  • Avisarme cuando se detecten actividades de rastreo: marque esta casilla para que se le muestre una barra de notificación cuando se bloquee contenido que puede utilizarse para rastrear. Esta barra contiene los botones siguientes:
    • Seguir bloqueando: ocultar la barra de notificación sin cargar el contenido desde un sitio de rastreo identificado.
    • Desbloquear: permitir que se cargue el contenido ahora y automáticamente cuando se visite esta página de nuevo en el futuro.
    • Preferencias: abre el panel de preferencias de privacidad y seguridad.

Notas:

  • Respetar una solicitud Do not track es voluntario, por lo que los sitios web no están obligados a obedecerla. Los sitios y servicios web que siguen la política deberían dejar de recopilar y usar información sobre su patrón de navegación web para toda la actividad para la cual se selecciona Do Not Track.
  • Solicitar Do not track de un sitio web no debería afectar a su capacidad de usar el sitio web correctamente (p.e., al usar una aplicación web o al añadir elementos en un carrito mientras compra).
  • Observe que Do Not Track no impide que se le muestren anuncios mientras navega. Puesto que el sitio web no tiene permitido recopilar y utilizar información sobre usted, el contenido de los anuncios serán menos específicos si no tiene la opción Do Not Track seleccionada.
  • La prevención de rastreo bloquea activamente contenido conocido por ser usado para el rastreo de una página web. Por tanto, la página puede aparecer de manera incompleta. Tales sitios son identificados descargando periódicamente una lista de sitios de rastreo conocidos que serán bloqueados si se activa. Se continuarán cargando los sitios de rastreo que no estén en esa lista hasta que sean incorporados a la misma.
  • La selección Desbloquear en la barra de notificación para un sitio específico puede revocarse en la pestaña Permisos del administrador de datos. Cuando se encuentra en una ventana privada, estas opciones no están disponibles en la barra de notificaciones.

Navegación basada en ubicación

Los sitios web pueden querer obtener información más detallada sobre la ubicación del usuario, p.e., para ofrecer servicios específicos cercanos a la posición actual. Para ese propósito, pueden solicitar la longitud y latitud (junto con otra información como altitud y dirección, si está disponible) del navegador. &brandShortName; nunca proporcionará esa información sin el consentimiento del usuario. Sin embargo, se mostrará una notificación por la página que solicita la ubicación al menos la primera vez que se visita este sitio. Las preferencias situadas en la sección central del panel de privacidad y seguridad le permiten desactviar por completo esta característica:

  • Solicitarme permiso si se hace una petición: seleccione esta opción si quiere que &brandShortName; le pregunte si el sitio web que está visitando solicita información de geolocalización. La notificación proporciona las siguientes opciones:
    • Compartir ubicación: seleccione esta opción si quiere que se determine la ubicación actual y se le tranmista al sitio solicitante, solo para esta solicitud.
    • No para esta solicitud: seleccione esta opción si no quiere que se determine la ubicación actual ni se le transmita al sitio solicitante, pero que se vuelva a preguntar en la próxima solicitud.
    • Siempre para este sitio: seleccione esta opción si quiere que se determine la ubicación actual y se le transmita al sitio solicitante, también para todas las solicitudes futuras.
    • Nunca para este sitio: seleccione esta opción si no quiere que se determine la ubicación actual ni se le transmita al sitio solicitante, tampoco en todas las solicitudes futuras.
    Cerrar u ocultar las notificaciones no dará ninguna respuesta a la página solicitante.
  • Desactivar esta característica y denegar todas las solicitudes: seleccione esta opción para que no se le vuelva a preguntar por solicitudes de geolocalización. &brandShortName; aparecerá ante el sitio web como si no admitiera esta característica.

Notas:

  • La geolocalización es un servicio web de terceros. &brandShortName; recopila cierta información sobre su acceseo a internet (p.e., puntos de acceso inalámbrico) y los envía a este servicio.
  • La precisión de la respuesta varía y puede oscilar desde el nivel de ciudad hasta unos pocos metros de su ubicación real.
  • No se proporciona información sobre el sitio solicitante al servicio de geolocalización. La información devuelta por ese servicio se proporciona al sitio web por &brandShortName; en respuesta a su solicitud.
  • El sitio solicitante recibe un token de identificación que le permite rastrear su movimiento durante un tiempo limitado usando el servicio de geolocalización.

Navegación segura

Algunos sitios web incluyen contenido potencialmente peligroso y pueden haber sido identificados como maliciosos. Las preferencias localizadas en la parte inferior del panel Privacidad y seguridad le permiten seleccionar qué tipos de sitios identificados de esta forma serán bloqueados. Si se activan &brandShortName; compara la dirección de cada sitio web que vaya a visitar contra listas actualizadas periódicamente. Se emite una advertencia previa a la carga real del sitio identificado, dándole de esta manera la oportunidad de cancelar el proceso de carga. En esta sección están disponibles las siguientes opciones:

  • Bloquear los sitios señalados como atacantes: marque esta casilla si quiere que &brandShortName; le avise si el sitio web que está a punto de visitar ha sido identificado como uno que intenta infectar su equipo con software malicioso. Este tipo de malware puede interferir con las funciones de su equipo o robar su información personal. También es usado frecuentemente para enviar correo spam o para distribuir más malware.
  • Bloquear los sitios señalados como falsificaciones web: marque esta casilla si quiere que &brandShortName; le avise si el sitio web que está a punto de visitar ha sido identificado como un sitio que finge ser un sitio web legítimo de alguna organización o servicio conocidos. Los sitios web como este se crean a menudo para engañarle para que proporcione un nombre de usuario y una contraseña u otra información personal confidencial (conocido normalmente como phishing).

Cuando intenta visitar un sitio web que ha sido identificado como malicioso (y esta característica está activada), verá una de las siguientes advertencias:

  • ¡Página identificada como ataque! Este sitio web ha sido identificado como un sitio web que intenta infectar su equipo con malware.
  • ¡Falsificación web identificada! Este sitio web ha sido identificado como un sitio web que intenta robar tu información personal.

En este punto no se ha producido ningún daño. Seleccione una de las opciones siguientes:

  • ¡Sácame de aquí! No cargar este sitio web. En su lugar, dirigirse a la página de inicio por defecto de &brandShortName;.
  • ¿Por qué se ha bloqueado esta página? Abre una página web que le muestra la información enviada sobre este sitio web concreto (ésta es una página web ubicada en el proveedor de este servicio). Si no existen estos datos del informe, se muestra una página con información general.
  • Ignorar esta advertencia: continuar con la carga de la página. Use esta opción sólo cuando esté seguro de que la página seleccionada es auténtica y visitarla sea seguro. Aparecerá una barra de notificación en la parte superior de la página, dándole la opción de informar al proveedor de este servicio en caso de que se trate de un error:
    • Éste no es un sitio atacante… o Ésta no es una falsificación web…: pulse este botón para abrir una nueva pestaña que le permita informar de advertencias erróneas (ésta es una página externa).

Importante: la característica de navegación segura está activada por defecto. No se recomienda desactivar ninguna de las dos funciones ya que puede dar lugar a visitar sitios no seguros.

Nota: si no se muestra una advertencia para un sitio web, esto no garantiza que un sitio sea confiable; sólo quiere decir que no se ha informado de él. Los sitios web maliciosos pueden parecer seguros pero estar intentando atacarle o robar su información personal en secreto. El propietario del sitio web o su proveedor pueden no saber que el sitio ha sido comprometido.
