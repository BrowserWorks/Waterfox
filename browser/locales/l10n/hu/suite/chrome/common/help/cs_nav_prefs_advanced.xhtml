Haladó beállítások - Find As You Type

This section describes how to use the Find As You Type references panel. If you're not already viewing it, follow these steps:

  1. Open the &brandShortName; Edit menu and choose Preferences.
  2. Under the Advanced category, click Find As You Type. (If no subcategories are visible, double-click Advanced to expand the list.)

The Find As You Type preferences panel allows you to control how you use the keyboard to search for text in web pages:

  • Find automatically when typing within a web page: If checked, typing text in a web page automatically activates Find As You Type and locates the text you typed (if it exists in the page). Choose whether you want typing to find any text in the page or links only. If unchecked, you must choose Find Links As You Type or Find Text As You Type from the Edit menu before typing the text you want to find.
  • Play a sound when typed text isn't found: If checked, Find As You Type plays a sound when the typed text isn't found in the web page.
  • Clear the current search after a few seconds of inactivity: If checked, the search will be cancelled after a few seconds of keyboard inactivity.
  • Show the find toolbar during find as you type: If checked, the find toolbar will open when Find As You Type is activated and what you type will be entered into the toolbar search field. If unchecked, the find toolbar will not be opened and the search string will be displayed in the status bar. Note that international text entry will not work in this mode.

Speciális proxybeállítások

Amennyiben különféle protokollokhoz különféle proxykiszolgálókat szeretnénk használni vagy ha SOCKS proxykiszolgálót kell beállítanunk:

  • HTTP-proxy, SSL-proxy, FTP-proxy, Gopher-proxy: írjuk be a proxykiszolgáló nevét vagy numerikus IP-címét. Adjuk meg a portot a Port mezőben.

    Figyelmeztetés: ha minden proxy-típushoz azonos beállításokat használunk, válasszuk ki a A HTTP-proxy beállításainak használata minden protokollhoz jelölőnégyzetet.

  • SOCKS-kiszolgáló: írjuk be a proxykiszolgáló nevét vagy numerikus IP-címét. Adjuk meg a portot a Port mezőben.
    • SOCKS v4, SOCKS v5: SOCKS-kiszolgáló megadásakor a használt SOCKS verziótól függõen válasszuk a SOCKS v4 vagy a SOCKS v5 beállítást.
    • Használat gépnevek feloldására (javasolt a SOCKS v5-nél): válasszuk ki ezt a jelölőnégyzetet, ha azt szeretnénk, hogy a SOCKS proxykiszolgálót használjuk a gépnevek feloldására. Ez az ajánlott beállítás SOCKS v5 proxykiszolgálók esetén.

Haladó beállítások – HTTP-beállítások

Ez a témakör a HTTP-beállítások panellap beállításait ismerteti. A párbeszédpanel a következő lépésekkel hívható elő:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. A Haladó kategóriában kattintsunk a HTTP-beállítások elemre. (Ha nem láthatóak a kategória beállításai, a lista kibontásához kattintsunk duplán a Haladó kategóriára.)

A HTTP-beállítások panellapon a HTTP alapú hálózati beállítások adhatók meg:

  • Közvetlen kapcsolat beállításai, Proxyn keresztüli kapcsolat beállításai: adjuk meg a HTTP verzióját, valamint a közvetlen és a proxykiszolgálón keresztüli kapcsolat beállításait.
    • HTTP 1.0 használata: válasszuk ezt a beállítást a HTTP eredeti, 1996-ban szabványosított verziójának használatához.
    • HTTP 1.1 használata: jelöljük be ezt a beállítást a HTTP új, nagyobb teljesítményű verziójának használatához, amely a kapcsolatok és az ügyfél oldali gyorsítótár hatékonyabb kihasználását, a többszörös HTTP-kérelmek kezelését és a házirendek jobb kézbentartását biztosítja.
    • Tartós kapcsolatok (keep-alive) engedélyezés: a beállítás választásakor a kapcsolat megnyitva marad, így további HTTP-kérelmek fogadhatók, ami növeli a sebességet.
    • Adatcsatornák (pipeling) engedélyezése: bejelölésével az adatcsatornák használatát engedélyezzük, ami egyidejűleg több HTTP-kérelem elküldését teszi lehetővé a kiszolgálóba, így csökkenthető a weboldalak betöltési ideje.

      Megjegyzés: az adatcsatornák használata csak a HTTP 1.1-es verziójában áll rendelkezésre.

  • User Agent String: The identifier sent by &brandShortName; to all websites is used for statistics about website usage but also sometimes to expose certain features only to known browsers (a practice known as "sniffing").
    • Advertise Firefox compatibility: If this is enabled, &brandShortName; will identify itself as both &brandShortName; and also compatible with Firefox. This allows websites that check for certain browsers rather than certain functionality to work with &brandShortName;.

Haladó beállítások – Egérgörgő

Ez a témakör az Egérgörgő panellap beállításait ismerteti. A párbeszédpanel a következő lépésekkel hívható elő:

Az egérgörgő beállításai párbeszédpanel segítségével testreszabhatja az egérgörgő viselkedését a &brandShortName; programban. Az egérgörgőt egyes egereknél az egér két gombja között találjuk meg. A modern egereknek két görgőjük is lehet, vagy egy kapcsoló segtségével válthatunk a függőleges és vízszintes görgetési irány közül. A függőleges görgetésre vonatkozó beállításokat a párbeszédpanel felső részén a Függőleges görgetés résznél találja meg, míg a vízszintes görgetés tulajdonságait az alsó rész szabályozza amelyet Vízszintes görgetés néven találhatunk meg.

  • Válasszuk ki a Nincs módosító billentyű lehetőséget, vagy adjuk meg az általunk - az egérgörgővel használni kívánt - billentyűt. A jelölőnégyzetek használatával az alábbi beállítások elvégzésére nyílik lehetőségünk:
    • Dokumentum görgetése: bejelölésekor a dokumentum görgetése a mezőben megadott számú sorral történik.
      • Az alapértelmezett rendszerbeállítás használata: választásakor a megadott sorszám helyett a rendszer alapbeállítását használja.
    • Görgetés egy oldallal feljebb vagy lejjebb: bejelölésekor oldalanként történik a görgetés. Ez az oldal gyorsabb, de pontatlanabb görgetését eredményezi.
    • Előre/Vissza az előzmények közt: választásakor az egérgörgő a korábban meglátogatott oldalak közötti böngészésre használható.
    • Szövegméret növelése/csökkentése: bejelölésekor az egérgörgő a weboldalon a betűméret növelésére vagy csökkentésére használható. Ezzel a szöveg tehető olvashatóbbá, illetve a teljes oldal megjeleníthető a képernyőn.

    Megjegyzés: minden módosító billentyűt különböző funkciókhoz rendelhető.

  • Amennyiben az egerünk nem rendelkezik vízszintes görgetési móddal, úgy az alsó - Vízszintes görgetés - panelen elvégzett minden beállítás figyelmen kívül lesz hagyva.

Amikor kiválasztunk egy elemet a weboldalon a DOM Inspector segítségével, az automatikusan villogtatással vagy keret megjelenítésével kijelölésre kerül:

    • Szegély színe: Adjuk meg az elem körül megjelenő keret színét.
    • Szegély vastagsága: Adjuk meg az elem körül megjelenő keret vastagságát.
    • Villogtatás időtartama: Adjuk meg a villogtatás időtartalmát (ezredmásodpercben megadva), ameddig a keret villogni fog.
    • Villogtatás sebessége: Adjuk meg két felvillanás közötti időtartamot (ezredmásodpercben megadva).
    • Inverz színek: Jelöljük be azt a jelölőnégyzetet, ha azt szeretnénk, hogy a kiválasztott elem inverz szinekkel jelenjen meg. Ezen beállítás hatására az egész elem —kerettel együtt— villogni fog.

Haladó beállítások – Rendszer

Ez a témakör a Rendszer panellap beállításait ismerteti. A párbeszédpanel a következő lépésekkel hívható elő:

  1. Válasszuk a Szerkesztés menü Beállítások parancsát.
  2. A Haladó kategóriában kattintsunk a Rendszer elemre. (Ha nem láthatóak a kategória beállításai, a lista kibontásához kattintsunk duplán a Haladó kategóriára.)

A párbeszédpanelen a &brandShortName; megnyitandó fájlok és protokollok adhatók meg:

  • A Windows a &brandShortName;-t fogja használni a következő fájltípusok megnyitásához: jelöljük ki azokat a fájltípusokat, amelyeket alapértelmezés szerint a &brandShortName;-val szeretnénk megnyitni .
  • A Windows a &brandShortName;-t fogja használni a következő protokollokhoz: jelöljük ki azokat a protokollokat, amelyeket alapértelmezés szerint a &brandShortName;-val szeretnénk megnyitni.
  • Figyelmeztessen, ha egy másik alkalmazás megváltoztatja ezeket a beállításokat: bejelölésekor a &brandShortName; üzenetet jelenít meg, ha másik alkalmazás megváltoztatja a fájltípusok és protokollok megnyitására vonatkozó beállításokat.

Tipp: A &brandShortName; alapértelmezett böngészőként való beállításáról lásd Böngésző beállításai - Böngésző. Also select A Figyelmeztessen, ha egy másik alkalmazás megváltoztatja ezeket beállításokat négyzet bejelölésekor arról is értesítést kapunk, ha a &brandShortName; helyett másik program lesz az alapértelmezett böngésző.
