Gyorsbillentyűk használata

A dokumentumban a következő formában vannak felsorolva a gyorsbillentyűk:

Például:

Parancs Windows Mac OS Linux vagy Unix
Másolás Ctrl+C Cmd+C Ctrl+C

A parancs végrehajtásához nyomjuk le egyszerre a megadott billentyűket. Például a másoláshoz Windowsban nyomjuk le egyszerre a Ctrl és a C billentyűt.

Tipp: A pluszjel (+) azt jelenti, hogy a két billentyűt egyszerre kell lenyomni. Ha egynél több pluszjel van (például Ctrl+Shift+C), az azt jelenti, hogy a pluszjeleket körülvevő mindhárom billentyűt egyszerre kell lenyomni.

Rövidítések:

  • Ctrl = Control billentyű
  • Cmd = Command billentyű a Macintosh billentyűzeten

Megjegyzés: Bizonyos gyorsbillentyűknek a kurzorpozíciótól (fókusz) függően különböző hatásuk van. Ha például Windows környezetben egy weboldal olvasása közben lenyomjuk a Home billentyűt, a &brandShortName; az oldal tetejére ugrik. Ha viszont Windows környezetben egy szövegmezőben állva nyomjuk le a Home billentyűt, a kurzor a szövegmező elejére ugrik.
