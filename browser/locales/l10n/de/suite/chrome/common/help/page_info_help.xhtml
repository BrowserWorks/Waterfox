Seiteninformationen anzeigen

Die Seiteninformationsdialogbox besteht aus mehreren Kategorien (Tabs) die verschiedene Informationen über die zur Zeit besuchte Webseite anzeigen. Diese Abschnitte zeigen einen Überblick über die einzelnen Kategorien und deren Informationen.

Um sich die Seiteninformationen über die aktuelle Webseite anzeigen zu lassen, öffnen Sie das Menü Ansicht und klicken dort auf Seiteninformationen. Dort können Sie dann die entsprechende Kategorie für die benötigten Informationen auswählen.

Die meisten Informationen die Sie unter den Kategorien Allgemein, Formulare, Links und Medien finden, werden aus dem Quelltext der HTML-Seite geholt. Diese Informationen sind normalerweise nur wür Webdesigner und andere Spezialsisten notwendig.

Für detailiertere Informationen über HTML, inklusive der angezeigten Tags in den Seiteninformationen lesen Sie HTML 4.01 Spezifikationen.

In diesem Abschnitt:
  • Allgemein-Tab
  • Formulare-Tab
  • Links-Tab
  • Medien-Tab
  • Datenschutz-Tab
  • Sicherheits-Tab

Allgemein (Seiteninformations Tab)

Wenn Sie die Seiteninformationen aus dem Ansichts-Menü aufrufen, sehen Sie zuerst das Allgemein-Tab, welches grundlegende Informationen zur gerade angezeigten Webseite darstellt.

Oben sehen Sie den Namen der Webseite (falls es einen gibt) und die folgenden Informationen

  • URL: Der Uniform Resource Locator dieser Webseite. Dies ist die standardisierte Adresse, die Sie auch in der Adresszeile des Browsersfensters sehen können.
  • Anzeigemodus: Zeigt an ob der Browser den Kompatibilitätsmodus (Quirks) oder den Standardkonformen Modus zum Darstellen der Webseite benutzt. Der Quirks-Modus ist zur Darstellung von nicht standardisierten Webseiten (zB: Webseiten die schon älter sind und sich an ältere Browser richten). Der Standardkonforme Modus hält sich hingegen an die striken Spezifikationen. Mozilla verwendet je nach Quelltext den richtigen Render-Modus.
  • Quelle: Hier sieht man ob die Webseite aus dem Cache oder nicht abgerufen wird.
  • Kodierung: Gibt den verwendeten Zeichensatz für dieses HTML-Dokument an.
  • Größe: Zeigt, falls verfügbar, die Größe der Datei an.
  • Modifiziert: Zeigt das letzte Änderungsdatum an (falls verfügbar).
  • Läuft ab: Das Datum an dem die Information auf dieser Webseite ab läuft wird.

Der untere Abschnitt zeigt die verwendeten Metatags der Webseite an. Metatags geben Auskunft über die Art des Inhalts dieser Webseite. Dabei handelt es sich um Informationen wie eine generelle Beschreibung zur Webseite, Schlüsselwörter für Suchmaschinen, Copyright Information usw.

Formulare (Seiteninformations Tab)

Wenn Sie die Seiteninformationen aus dem Ansichts-Menü aufrufen und auf das Formulare-Tab klicken, sehen Sie Informationen über die verwendeten Formulare auf dieser Webseite.

Im oberen Teil sehen Sie grundlegende Informationen über die Art wie jedes Formular in den HTML-Quelltext integriert wurde.

  • Name: Falls vorhanden der Formularname.
  • Methode: Die benutze HTML-Methode, wie die Informationen des Formulars an den Webserver übermittelt werden. GET hängt die eingebenen Daten des Formulars an die zu schickende Adresse. POST schickt die Daten als Parameter an eine Webseite, wo Sie von einem Programm weiter verarbeitet werden können.
  • Aktion: Die URL des Programms, welches durch das Formular angesteuert wird.

Wenn Sie eines der aufgelisteten Formulare des oberen Teils auswählen, werden in der unteren Hälfte genauere Informationen über die einzelnen verwendeteten Formularelemente angezeigt:

  • Beschreibung: Die Beschreibung (falls angegeben) des Elements.
  • Feldname: Der Name des Elements.
  • Typ: Der Eingabetyps des Elements, wie TEXT (für Text), RADIO (für einen Radio-Button) oder HIDDEN (für gespeicherte Information die zwar übermittelt aber nicht am Bildschirm angezeigt wird).
  • Aktueller Wert: Der aktuelle Wert des Elements. Zum Beispiel ist der aktuelle Wert eines Text-Elements der Text den es derzeit enthält.

Links (Seiteninformations Tab)

Wenn Sie die Seiteninformationen aus dem Ansichts-Menü aufrufen und auf das Links-Tab klicken, sehen Sie eine Liste aller Links auf dieser Webseite. Die folgenden Informationen werden für jeden Link angezeigt:

  • Name: Der Text der im Browser als Link erscheint.
  • Addresse: Die URL der Webseite, auf die der Link verweist.
  • Typ: Die Art des Links, wie Anker (für einen Link zu einer speziellen Stelle in einem HTML-Dokument) oder Formular absenden.

Medien (Seiteninformations Tab)

Wenn Sie die Seiteninformationen aus dem Ansichts-Menü aufrufen und auf das Medien-Tab klicken, sehen Sie eine Liste aller Grafiken und anderer Medienelemente dieser Webseite.

Der obere Teil listet die Adresse (URL) und den Typ jedes Elements auf.

Wenn Sie nun ein Medienelement aus dem oberen Teil markieren, sehen Sie im unteren Abschnitt verfügbare Informationen wie Beschreibung, Größe oder Dimensionen dieses Elements.

Außerdem sehen Sie das ausgewählte Element am unteren Ende der Dialogbox. Um größere Bilder anschauen zu können, können Sie das Seiteninformationsfenster durch klicken in der rechten unteren Ecke vergrößern (klicken und ziehen).

Um ein Medienelement als seperate Datei zu speichern:

  • Speichern unter...: Wählen Sie das Element das sie speichern wollen und klicken Sie auf Speichern unter... und wählen Sie das Verzeichnis in dem Sie die Datei speichern wollen.

Datenschutz (Seiteninformations Tab)

Wenn Sie sich die Seiteninformationen ansehn, klicken Sie auf Datenschutz um eine Liste von Elementen zu sehen, welche Teil der Webseite sind.

  • Seitenkomponenten: Unter der Adresse der Webseite sind einige Kategorien. Um eine Liste der Kategorien zu sehen, doppelklicken Sie auf dessen Name:
    • Links: Zeigt alle Links.
    • Java-Applets: Zeige eine Liste der Java-Applets (sofern es welche gibt).
    • Image-Maps: Zeige die gesamten Image-Maps.
    • Formulareingaben: Zeige eine Liste der Scripts welche von der Seite genutzt werden um Daten anzufordern.
    • Frames: Zeige die Frames auf der Webseite.
    • Grafiken: Zeige alle Grafiken, welche auf der Webseite benutzt wurden.
    • Extern geladene Dokumente: Zeigt eine Liste der externen Dokumente, welche zur Anzeige der Seite geladen sind (wie zum Beispiel CSS-Dateien oder Grafiken).
    • Generische Objekte: Zeigt eine Liste der generischen Objeckte.
    • Skripte: Zeigt eine Liste aller Scripte auf der Webseite.
  • Richtlinien: Diese Schaltfläche versucht, die Datenschutz-Richtliniendatei der Webseite zu öffnen.
  • Zusammenfassung: Wählen Sie diese Schaltfläche um eine Zusammenfassung der Datenschutz-Richtlinien der aktuellen Site zu erhalten.
  • Optionen: Beim Klick auf diese Schaltfläche wird versucht, ein Fenster zu öffnen, in welchem Sie Ihre persönlichen Daten welche auf der Webseite gespeichert sind, in welchem Sie diese ändern können.

Sicherheit (Seiteninformations Tab)

Wenn Sie die Seiteninformationen aus dem Ansichts-Menü aufrufen und auf das Sicherheits-Tab klicken, sehen Sie Informationen über die Authentifikation und Verschlüsselung dieser Webseite. Der obere Teil zeigt ob die Identität der Webseite verifiziert wurde, während der untere Teil angibt ob die Webseite verschlüsselt war, als Sie der Browser erhielt.

Sie können das Sicherheits-Tab auch direkt öffnen, indem Sie auf das Schloß-Symbol in der rechten unteren Ecke Ihres Browsers klicken.

Der obere Teil des Sicherheits-Tabs kann folgendes anzeigen:

  • Website-Identität nicht verifiziert_ Die Webseite die Sie zur Zeit besuchen, besitzt kein Zertifikat um sich selbst zu authentifizieren. Deswegen kann der Zertifikat-Manager die Identität dieser Webseite nicht verifizieren. Es ist möglich, obwohl unwahrscheinlich, dass diese Webseite nicht das ist was sie zu seien scheint.
  • Website-Identität verifiziert: Das Zertifikat das der Zertifikats-Manager für die Identität dieser Webseite verwendet hat, wurde von einer Zertifizierungsstelle (CA) bestätigt. Sie können sich sicher sein, dass diese Webseite das ist was sie vorgibt zu sein.

Der untere Abschnitt des Sicherheits-Tabs kann eine der folgenden Meldungen anzeigen:

  • Verbindung nicht verschlüsselt: Es ist möglich das andere Leute Zugriff auf die gesendeten und empfangenen Informationen zwischen Computer und Webseite erlangen können (wenn auch unwahrscheinlich).
  • Verbindung verschlüsselt: Im allgemeinen basiert die Stärke der Verschlüsselung auf die Länge des Schlüsseln gemessen in Bit. Umso länger der Schlüssel ist, umso stärker ist auch die Verschlüsselung. Dies wiederum bedeutet, dass das Entschlüsseln durch eine unauthorizierte Person immer schwieriger wird.

    Im Sicherheits-Tab wird die Verschlüssungsstärke auf drei Arten angegeben:

    • hochgradige Verschlüsselung: Stärkste verfügbare Verschlüsselung mit 128-bit Schlüsseln als Minimum.
    • mittelgradige Verschlüsselung: Um einiges stärker als eine niedrigere Verschlüsselung. Hierbei werden Schlüssellängen von 56- oder 64-Bit verwendet.
    • niedrige Verschlüsselung: Schwächste verfügbare Verschlüsselung mit 40-Bit Schlüsseln.

    Viele Webseiten unterstützen eine hochgradige Verschlüsselung. Wenn sie auf Seiten mit einer schwächeren Verschlüsselung sind, ist es möglich die gesendeten und empfangenen Infomrationen abzuhören. Allerdings ist es sehr unwahrscheinlich.

Das nicht vorhanden sein einer Verschlsüüelung oder einer hochgradigen Verschlüsselung, sollte Ihnen bei vertraulichen Daten wie Kreditkartennummern zu denken geben.

Sie können schnell den Verschlüsselungsstatus einer Webseite anhand des Schloß-Symbols in der rechten unteren Ecke des Browsers kontrollieren. Für mehr Informationen lesen Sie bitte Sicherheit einer Webseite kontrollieren.
