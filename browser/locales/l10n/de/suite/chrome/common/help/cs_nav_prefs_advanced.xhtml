  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.

Im Abschnitt Scripts & Plugins wird das Verhalten von JavaScript in &brandShortName; festgelegt.

  • JavaScript aktivieren:
    • Browser: Um dem Browser die Ausführung von JavaScript in Webseiten zu erlauben.
  • Scripts folgende Aktionen erlauben: Wählen Sie die Checkboxen um festzulegen, wie Javascript genutzt werden darf:
    • Existierende Fenster verschieben oder deren Größe ändern: Erlaubt das Verschieben und das Ändern der Größe von Fenstern durch Javascript.
    • Fenster vor oder hinter andere Fenster legen: Erlaubt, Fenster über oder unter andere Fenster zu legen.
    • Statusleiste verstecken: Erlaubt die Statusleiste auszublenden.
    • Statuszeilentext ändern: Erlaubt die Änderung des Statuszeilentextes, z.B. in eine Laufschrift in der Statuszeile.
    • Grafiken verändern: Erlaubt die Änderung oder Animation von Bildern, z.B. Rollovers (Bilder, die sich Ändern, wenn der Mauszeiger darüber platziert wird).

    • Mail & Newsgroups: Erlaubt Plugins in Mail & Newsgroups verwendet zu werden.
  • Wenn zusätzliche Plugins benötigt werden: Aktivieren Sie diese Checkbox wenn Sie darüber informiert werden wollen, falls eine Webseite zusätzliche Plugins benötigt
    • Eine Benachrichtigungsleiste am oberen Rand des Inhaltsbereichs anzeigen: Wenn eine Webseite ein noch nicht installiertes Plugin benötigt, wird oberhalb der Webseite eine Benachrichtigungsleiste eingeblenden. Erweiterte Einstellungen - Suche beim Tippen

Die Einstellungs-Seite für Suche beim Tippen erlaubt es Ihnen, wie die Tastatur beim Suchen von Text auf einer Web-Seite verwendet werden soll:

  • Beim Eintippen innerhalb einer Webseite automatisch suchen: Wenn gewählt können Sie durch Eintippen eines Textes diesen automatisch in dieser Webseite suchen. Sie können hierbei auswählen, ob Sie jedlichen Text oder nur Links suchen wollen. Wenn Sie diese Option deaktiveren und die Funktion trotzdem nutzen wollen, müssen Sie im Menü Bearbeiten&brandShortName; die Schaltflächen Links beim Tippen suchen oder Text beim Tippen suchen benutzen.
  • Einen Klang apspielen, wenn der eingetippte Text nicht gefunden wurde: Wenn gewählt wird ein Klang abgespielt, wenn der gesuchte Text in dieser Webseite nicht gefunden wurde.
  • Die aktuelle Suche nach einigen Sekunden Inaktivität beenden: Wenn ausgewählt, wird die aktuelle Suche nach einigen Sekunden Tastatur-Inaktivität beenden.
  • Die Suchleiste beim Suchen während der Eingabe anzeigen: Wenn aktiviert wird die Such-Leiste eingeblendet sobald Sie einen Text auf einer Webseite mittels der Funktion Suche beim Tippen suchen. Der von Ihnen gesuchte Begriff wird in die Such-Box der Leiste eingefügt. Wenn Sie diese Funktion deaktivieren, wird die Such-Leiste nicht angezeigt. Der gesuchte Begriff wird jedoch in der Status-Zeile angezeigt. Beachten Sie, das die Suche nach Internationalen Texten auf in diesem Modus nicht funktioniert.

  1. Öffnen Sie das Menü Bearbeiten&brandShortName; und wählen Sie Einstellungen
  2. Wählen Sie in der Kategorie Erweitert Cache (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Erweitert).

Die Kategorie Cache legt fest, wie Mozilla Webseiten im Speicher oder auf Festplatte zwischenspeichert:

  • Größe des Caches von &brandShortName; verwalten lassen: Aktivieren Sie diese Option, wenn &brandShortName; die Größe des Caches anhand des vorhandenen Speicherplatzes Ihres Datenträgers verwalten soll (dies ist die Standard-Einstellung)
  • [__] MB am Datenträger für den Cache verwenden: Geben Sie hier die Gewünschte Größe ein, welche &brandShortName; für den Cache reservieren soll. Der Datenträger-Cache kann wiederverwendet werden, auch wenn Sie den Computer neu starten. (Die Standard-Größe ist 1024 MB. Diese Einstellung wird ignoriert, wenn die Option darüber aktiviert ist.)
  • Cache löschen: Klicken Sie, um den Festplatten-Cache zu löschen.
  • Cache-Ordner Standord: Zeigt den Ordner auf der Festplatte, der für Cache-Dateien verwendet wird
    • Ordner wählen: Klicken Sie, um den Ordner für den Cache zu ändern.
  • Vergleich zwischen Cache-Dokument und Netzwerk-Dokument erfolgt:
    • Jedes Mal: Wählen Sie diese Option, wenn &brandShortName; bei jedem Seitenaufruf die Webseite mit der im Cache gespeicherten Seite vergleichen soll.
    • Wenn die Seite veraltet ist: Wählen Sie diese Option, wenn &brandShortName; die Webseite mit der im Cache gespeicherten Seite vergleichen soll, wenn die Seite vom Server als veraltet bezeichnet wird.
    • Einmal pro Sitzung: Wählen Sie diese Option, wenn &brandShortName; bei jedem Neustart von &brandShortName; die Webseite mit der im Cache gespeicherten vergleichen soll.
    • Nie: Wählen Sie diese Option, wenn Sie nicht wollen, dass &brandShortName; die Webseite mit der im Cache gespeicherten Seite vergleicht.
  • Links im Voraus laden: Wählen Sie diese Option, wenn sie Links auf andere Webseiten, welche im voraus geladen werden sollen während Sie diese Seite ansehn im Hintergrund vorladen wollen, sodass bei Klick auf den Link diese Seite schneller angezeigt werden kann. Für mehr Informationen über Prefetching, siehe Link Prefetching FAQ.

    • SOCKS v4, SOCKS v5: Wenn Sie einen SOCKS Proxy verwenden wählen Sie entweder SOCKS v4 oder SOCKS v5, abhängig davon, welche SOCKS-Version Ihr Proxy versteht.
    • Zur Hostname-Auflösung verwenden: Wählen Sie diese Option, wenn Sie Ihren SOCKS-Poxy zur Hostname-Auflösung verwenden möchten. Diese Einstellung wird für SOCKS v5 Proxys empfohlen.

Erweiterte Einstellungen - HTTP-Verbindungen

Dieser Abschnitt beschreibt die Verwendung von HTTP-Verbindungen. Falls Sie den Abschnitt HTTP-Verbindungen noch nicht vor sich sehen, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen
  2. Wählen Sie in der Kategorie Erweitert HTTP-Verbindungen. (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Erweitert).

Der Abschnitt HTTP-Verbindungen erlaubt Ihnen die Konfiguration der HTTP-Verbindungen zu bearbeiten:

  • Optionen für Direkte Verbindung, Optionen für Proxie-Benützung: Wählen Sie die HTTP-Version und die entsprechenden Optionen für Proxie- oder Direkte Verbindung.
    • HTTP 1.0 verwenden: Um die Orignalversion von HTTP zu verwenden, die 1996 standardisiert wurde.
    • HTTP 1.1 verwenden: Um die neuere HTTP-Version 1.1 zu verwenden, die bessere Leistung bietet (u.a. effizientere HTTP-Verbindungen, bessere Unterstützung von Client-seitigem Caching, mehrfache HTTP-Anfragen (sog. Piplining) und verbesserte Kontrolle über Nutzungsdauer und Ersatz von Cache-Inhalten).
    • Verbindungen aufrecht erhalten (Keep-Alive): Um Verbindungen aufrecht zu erhalten und für mehrere Anfragen die selbe Verbindung zu nutzen (erhöht die Geschwindigkeit).
    • Pipelining verwenden: Aktiviert Piplining, das die gleichzeitige Übertragung mehrerer HTTP-Anfragen ermöglicht, um Verzögerungen im Aufbau von Webseiten zu verringern.

      Hinweis: Pipelining ist nur unter HTTP 1.1 verfügbar.

  • User-Agent-String: Die Kennung welche von &brandShortName; an alle Webseiten sendet wird kann für eine Statistik der Webseiten-Benutzung verwendet werden. Teilweise werden bestimmte Funktionen nur für bekannte Browser aktiviert (dieser Vorgang wird sniffing genannt).
    • Firefox-Kompatibilität ausweisen: Wenn Sie diese Option aktivieren, wird &brandShortName; sich sowohl als &brandShortName; als auch kompatibel mit Firefox melden. Dies hilft das Webseiten, welche auf bestimmte Browser prüfen anstand auf die Unterstützung gewisser Funktionen, auch mit &brandShortName; verwendbar sind.

Erweiterte Einstellungen - DOM Inspector

Der DOM Inspector ist ein optionales Webentwicklungsprogramm. Dieser Abschnitt beschreibt, wie Sie dieses Einstellungs-Fenster nutzen. Falls Sie den Abschnitt DOM Inspector noch nicht vor sich sehen, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü&brandShortName; Bearbeiten und wählen Sie Einstellungen.
  2. Wählen Sie DOM Inspector in der Kategorie Erweitert (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Erweitert.)

Wenn Sie ein Element im DOM-Inspector auswählen, kann der DOM-Inspector dieses automatisch hervorheben indem dieser das Element oder dessen Rahmen blinken lässt:

  • Ausgewählte Elemente blinken lassen: Wählen Sie diese Option, um das gewählten Element hervorzueheben. Standardmässig wird ein blinkender Rahmen um das Element angezeigt.
    • Rahmenfarbe: Wählen Sie die Farbe des Rahmens welcher um das Element eingeblendet wird.
    • Rahmenbreite: Geben Sie die Breite des Rahmens ein.
    • Blinkdauer: Geben Sie die Zeitdauer für das Blinken ein (in Millisekunden).
    • Blinkgeschwindigkeit: Geben Sie die Anzahl von Millisekunden zwischen 2 Blinkabständen ein.
    • Farben invertieren: Wählen Sie diese Option um das ausgewählte Element in der invertierten Farbe des Rahmens zu zeichnen. Dies bewirkt, dass sowohl Rahmen aus auch das Element selbst blinkt.
