Webentwicklungsprogramme

Für Webentwickler stellt &brandShortName; einige Tools zur Verfügung die zur Entwicklung und zum Debugging von Webpapplikationen helfen sollen. Einige müssen extra installiert werden. Zu finden sind sie in dem Menü Extras unter Web-Entwicklung.

  • Fehler-Konsole: Ein Konsolen-Fenster welches Probleme mit JavaScript und CSS-Code in Web-Anwendungen sowie in &brandShortName; selbst anzeigt. Standardmässig werden Fehler beim Analysieren von CSS-Code sowie JavaScript-Fehler angezeigt, Sie können aber auch Meldungen aus XUL Und JavaScript-Code anzuzeigen.
  • DOM Inspector: Ein Tool zum Inspizieren und Editieren der DOM eines Dokuments oder der XUL Applikation.

    Online Ressourcen: (alle in englisch)

    • Introduction to the DOM Inspector (Ian Oeschger)
    • DOM Inspector FAQ (mozilla.org)
    • Creating Applications with Mozilla - Appendix B3: The DOM Inspector
  • JavaScript Debugger: auch bekannt als Venkman, ermöglicht dieses Programm das debuggen von JavaScript in &brandShortName;. Es unterstützt Stepping (Schrittweises durchgehen des Codes), Breakpoints und viele andere Funktionen.

    Online Ressourcen: (alle in englisch)

    • Venkman JavaScript Debugger Homepage
    • Learning the JavaScript debugger Venkman (Svend Tofte)
    • Debugging JavaScript Using Venkman (webreference.com, article series)
    • Venkman Development Homepage
