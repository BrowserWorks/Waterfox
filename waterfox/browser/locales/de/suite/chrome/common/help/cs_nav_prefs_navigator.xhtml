Browser-Einstellungen - Browser

Dieser Abschnitt beschreibt den Abschnitt Browser in den Einstellungen. Falls Sie den Abschnitt noch nicht vor sich sehen, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.
  2. Wählen Sie Browser.

Die Browser-Einstellungen erlauben das Konfigurieren von bestimmten Aspekten im Browser.

  • Anzeigen: Wählen Sie aus dem Drop-down Menü in Kombination mit den Kontrollkästchen, was beim Starten angezeigt werden soll und wann ein neues Fenster oder Tab geöffnet werden soll:
      • Wähle Bei Browser-Start um einzustellen was beim Start des Browsers angezeigt werden soll.
      • Wähle In Neuem Fenster um einzustellen was beim Öffnen eines neuen Fensters angezeigt werden soll.
      • Wähle In Neuem Tab um einzustellen was beim Öffnen eines neuen Tabs angezeigt werden soll.
    2. Wählen Sie eine der folgenden Optionen:
      • Leere Seite: Wenn &brandShortName; starten soll, ohne dass eine Seite geladen wird.
      • Startseite: Um mit Ihrer Starseite zu beginnen (wird weiter unten festgelegt).
      • Letzte angezeigte Seite: Um mit der Seite zu beginnen, die Sie zuletzt besucht haben.
      • Vorherige Sitzung wiederherstellen: Veranläst den Browser die Fenster und Tabs, welche beim letzen Mal als &brandShortName; geschloßen wurde, wiederherzustellen. Diese Option ist nur in Verbindung mit der entsprechenden Option unter Startseite verfügbar.
  • Standard-Browser: Erlaubt Ihnen &brandShortName; als Ihren Standard-Browser zu definieren oder zeigt das er es schon ist.
    • Standard-Browser setzen: Wenn nicht grau, klicken Sie auf den Button um &brandShortName; zum Standard-Browser zu machen.
  • Startseite: Geben Sie die Adresse der Seite ein, die Sie als Ihre Startseite verwenden wollen, oder wählen Sie eine der folgenden Optionen:
    • Aktuelle Seite verwenden: Klicken Sie hier um die aktuell im Browser angezeigte Seite als Startseite zu verwenden.
    • Aktuelle Gruppe verwenden: Haben Sie zwei oder mehr Browser-Tabs offen und wollen diese als Ihre Startseiten verwenden, so klicken Sie hier. Nach dem auswählen sehen Sie in dem Adressfeld "-- Startseiten-Gruppe ist gesetzt --" als Bestätigung.

      Achtung: Bearbeiten Sie das Adressfeld nach betätigen des Buttons, wird die Startgruppeneinstellung gelöscht.

    • Standard wiederherstellen: Wählen Sie diese Option um die Standardwebseite einzustellen.
    • Durchsuchen...: Wählen Sie diese Option um eine Datei auf Ihrer Festplatte als Startseite festzulegen.

Browser-Einstellungen - Hilfsanwendungen

Dieser Abschnitt beschreibt die Einstellungen zu den Hilfsanwendungen. Falls Sie den Abschnitt Hilfsanwendungen noch nicht vor sich sehen, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.
  2. Wählen Sie Hilfsanwendungen in der Kategorie Browser (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Browser).

In der Konfigurations-Maske für Hilfsanwendungen können auswählen, welches Programm oder sonstiger Händler für einen bestimmten Typ von Datei zuständig sein soll (z.B. PDF-Dokumente). Es zeigt Ihnen eine Liste der bekannten Datei-Typen und erlaubt es Ihnen die auszuführende Aktion zu konfigurieren. Um in dieser Liste zu filtern, können Sie das Such-Feld benutzen um den gewünschten Datei-Typ auszuwählen. In der Liste werden Ihnen dann jene Datei-Typen angezeigt, bei welchen entweder in der Typen-Beschreibung oder in der ausgewählten Aktion vorkommt.

Sie können ein lokales Program einstellen um den Datei-Typ zu verarbeiten. Für manche Typen können Sie auch eine Web-Applikation , ein Plugin in &brandShortName; wählen, oder die Datei auf Ihren Computer speichern.

Um einen Händler für einen bestimmten Typ zu setzen, wählen Sie den gewünschten Typ aus der Liste. Der Händler für diesen Typ verwandelt Sich dann zu einen Auswahl-Menü. Öffnen Sie das Menü und wählen den gewünschten Händler für diesen Typ. Abhängig vom ausgewählten Typ stehen Ihnen folgende Möglichkeiten zur Verfügung:

  • Wählen Sie ein Programm indem Sie diese im Menü auswählen. Wenn Sie ein Programm welches auf Ihrem PC installiert ist nutzen wollen, dieses jedoch nicht im Menü vorhanden ist, dann können Sie dieses auswählen indem Sie Andere verwenden… auswählen und das Programm auf Ihrem PC suchen.
  • Wählen sie ein Plugin durch Auswahl im Menü.
  • Speichern Sie alle Dateien dieses Typs auf Ihren PC indem Sie Save File im Menü wählen. Wenn Sie die Option Dateien automatisch in angegebenen Download-Ordner herunterladen in den Download-Einstellungen aktiviert haben, speichert &brandShortName; die Dateien automatisch auf Ihren PC. Ansonsten werden Sie nach dem Download-Ordner gefragt, sobald &brandShortName; Dateien dieses Typs findet.
  • &brandShortName; soll Immer nachfragen was Sie tun wollen, wenn ein Objekt des definierten Typs gefunden wird. Wenn Sie diese Option auswählen, bekommen Sie jedesmal eine Abfrage wenn Sie auf ein solches Objekt zugreifen und können auswählen, was sie tun möchten.

Hinweis: Wenn ein Plugin für einen Typ verfügbar ist und Sie einen andern Händler für diesen Typ auswählen, wird dieser Händler nur dann verwendet, wenn Sie das entsprechende File direkt aufrufen, wenn es in einer Webseite eingebunden ist, wird &brandShortName; das zugehörige Plugin nutzen, um den Inhalt darzustellen. Siehe dazu unter Plugins und Downloads.

Browser-Einstellungen - Adressleiste

Dieser Abschnitt beschreibt die Einstellungen zur Adressleiste. Falls Sie den Abschnitt Adressleiste noch nicht vor sich sehen, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.
  2. Wählen Sie Adressleiste in der Kategorie Browser (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Browser).

Die Adressleiste Einstellungseite lässt Sie das Verhalten der Adressleiste verfeinern.

  • Automatische Vervollständigung:
    • Bei der Eingabe automatisch mit besuchten Webseiten vervollständigen: Aktivieren Sie diese Option wenn &brandShortName; während einer Eingabe in die Adressleiste automatisch Vorschläge aus Ihrer Chronik anzeigen soll.
      • Nur zuvor eingegebene Webseiten durchsuchen: Zeigt nur Webseiten, welche Sie zuvor in der Adressleiste eingetippt haben und keine Seiten, welche Sie auf anderem Wege geöffnet haben, wie zum Beispiel durch Klick auf einen Link auf einer Webseite.
      • Nur in Adressen, nicht in Website-Titeln suchen: Zeigt nur Webseiten, wo der von Ihnen gesuchte Text in der Adresse vorkommt. Webseiten, wo der von Ihnen gesuchte Text im Seitentitel enthalten ist, werden nicht angezeigt, es sei den der gesuchte Text findet sich auch in der Adresse.
      • Übereinstimmung:
        • Überall in Adresse oder Titel: Es werden alle Webseiten angezeigt, in denen der gesuchte Text irgendwo im Titel oder der Adresse enthalten ist.
        • Überall, aber bevorzugt an Wortgrenzen: Es werden jene Adressen angezeigt, in denen der gesuchte Text irgenwo im Titel oder der Adresse enthalten ist. Vorkommen an Wort-Enden (dazu mehr im nächsten Punkt) werden jedoch bevorzugt. Dies ist die Standard-Einstellung.
        • Nur an Wortgrenzen: Es werden nur jene Adressen angezeigt, welche am Anfang eines Wortes im Title oder der Adresse vorkommen. Hierbei ist zu beachten, dass ein Grossbuchstabe innerhalb eines Wortes als Wortanfang gezählt wird.
        • Nur am Beginn von Adresse oder Titel: Es werden nur jene Webseiten angezeigt, in denen der gesuchte Text am Anfang der Adresse oder des Titels vorkommt.
      • Automatisch die beste Übereinstimmung eintragen: Während Sie in der Adressleiste einen Text eintippen, vervollständigt &brandShortName; automatisch die Adresse der am besten passenden Webseite in der Adressleiste.e Hinweis: Mit aktivieren dieser Option werden auch lokale Adressen (z.B. der Pfad zu Dateien auf Ihrer Festplatte) in der Adressleiste ausgefüllt, auch wenn Sie die Option Bei der Eingabe automatisch mit besuchten Webseiten vervollständigen deaktiviert haben.
      • Liste der übereinstimmenden Ergebnisse anzeigen: Während Sie in der Adressleiste einen Text eintragen, wird &brandShortName; Ihnen eine Liste aller besuchten Adressen anzeigen, welche zu Ihrem Suchtext passen. Hinweis: Wenn Sie die Option Bei der Eingabe automatisch mit besuchten Webseiten vervollständigen deaktiviert haben, werden Ihnen trotzdem lokale Adressen von Ihrer Festplatte in der Liste angezeigt.
  • Internet-Suchmaschine:
    • Standard-Suchmaschine anzeigen: Zeigt einen Listeneintrage in der Adressleiste, welcher es Ihnen erlaubt, den eingegebenen Text mit der Standard-Suchmaschine zu suchen.
  • Unbekannte Adressen
    • www. und .com an die Adresse anhängen, wenn eine Webseite nicht gefunden wird: Aktivieren Sie diese Option, wenn &brandShortName; automatisch www. an den Anfang und .com an das Ende der gesuchten Adresse anhängen soll, wenn diese nicht gefunden wird. Mehr Informationen über diese Funktion finden Sie auf dem Online-Dokument Domain Guessing (eng).
    • Eine Websuche durchführen, wenn der eingegebene Text keine Web-Adresse ist: Aktivieren Sie diese Option, wenn &brandShortName; automatisch eine Websuchen mit dem Text welchen Sie in der Adressleiste eingeben haben durchführen soll. Wenn der eingebene Text keine Webseite ist wird &brandShortName; automatisch eine Websuche beginnen, sobald Sie EnterReturn in der Adressleiste drücken. Hinweis: Die Suchmaschine kann nicht über die Konfiguration unter Internet-Suche geändert werden.

Legen die Einstellungen zum Tabbed Browsing fest:

      Tipp: Verwenden Sie die Beim Eintippen Suchen-Funktion um so schneller zum Link Ihrer Wahl zu kommen.

    • CmdStrg + ReturnEnter in der Adressleiste: Wählen Sie diese Option wenn Sie eine Webseite über die Adressleiste aufrufen und dabei CmdStrg + ReturnEnter, um sie in einem neuen Tab zu öffnen.

Browser-Einstellungen - Link Verhalten

Dieser Abschnitt beschreibt die Einstellungen des Link-Verhaltens von &brandShortName;. Falls Sie den Abschnitt Link Verhalten noch nicht geöffnet haben, führen Sie folgende Schritte aus:

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.
  2. Wählen Sie Link-Verhalten in der Kategorie Browser (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Browser).

Das Einstellungs-Fenster zum Link-Verhalten erlaubt es Ihnen zu konfigurieren wie Links in &brandShortName; geöffnet werden sollen:

  • Verhalten beim Öffnen von Links: Wenn eine Webseite einen Link in einem neuen Fenster öffnen will (sei es durch das Target-Attribut in HTML oder durch JavaScript) können Sie bestimmen, wie &brandShortName; diese Links öffnen soll:
    • Aktuellem Tab/Fenster: Öffnet die Seite im aktuellen Tab des aktiven Fensters.
    • Neuem Tab im aktuellen Fenster: Öffnet den Link in einem neuen Tab anstatt einem neuen Fenster.
    • Neuem Fenster: Öffnet die Seite in einem neuen Fenster (Dies ist die Standard-Einstellung.)
  • Links aus anderen Anwendungen: Wenn &brandShortName; aus anderen Programmen mit einer Webseite als Argument aufgerufen wird (wie zum, Beispiel durch einen Klick auf einen Link in Ihrem Email-Programm), können Sie kontrollieren, wie die Seite geladen werden soll:
    • Aktuellem Tab/Fenster: Öffnet die Seite im aktuellen Tab des aktiven Fensters.
    • Neuem Tab im aktuellen Fenster: Öffnet den Link in einem neuen Tab anstatt einem neuen Fenster.
    • Neuem Fenster: Öffnet die Seite in einem neuen Fenster (Dies ist die Standard-Einstellung.)

Wenn Sie wollen, daß &brandShortName; immer ein neues Tab öffnet, egal ob es gestartet wird oder bereits läuft, stellen Sie sicher, daß die folgenden Einstellungen gesetzt sind:

  • Wenn &brandShortName; mittels Kommandozeile gestartet wird, darf kein Kommandozeilen-Parameter (wie zum Beispiel -new-window) benutzt werden.
  • Unter Einstellungen Erscheinungsbild - Erscheinungsbild darf keine der Optionen für Öffne beim Start von &brandShortName; gesetzt sein.
  • Die Einstellung Links aus anderen Anwendungen muss auf Neuem Tab im aktuellen Fenster gesetzt sein.

Browser-Einstellungen - Downloads

  1. Öffnen Sie das Menü Bearbeiten &brandShortName; und wählen Sie Einstellungen.
  2. Wählen Sie Downloads in der Kategorie Browser (Wenn keine Unterkategorien sichtbar sind, doppelklicken Sie auf Browser).

Legt fest wie &brandShortName; mit Downlodas von Webseiten umgehen soll:

  • Beim Starten eines Downloads:
    • Den Download Manager in den Vordergrund bringen, wenn er bereits offen ist: Aktivieren Sie diese Option, wenn der Download-Manager den Fokus bekommen soll, wenn dieser bereits offen ist. Dieser zeigt Ihnen der Status der aktuellen und bereits beendeten Downloads in einem einzelnen Fenster. Diese Option wirkt nur, wenn der Download-Manager bereits offen ist und überschreibt die anderen Einstellungen.
    • Download-Manager öffnen: Wählen Sie diese Option um den Download-Manager anzuzeigen. Er wird nur angezeigt, wenn der Download-Manager derzeit noch nicht geöffnet ist.
    • Fortschritts-Dialog öffnen: Wählen Sie diese Option um einen Fortschritts-Dialog anzuzeigen, in welchem der Status des derzeitigen Downloads angezeigt wird. Jeder Download bekommt dabei seinen eigenen Fortschritts-Dialog.
    • Nichts öffnen: Wählen Sie diese Option wenn Downloads ohne Statusanzeige durchführen wollen.
  • Beim Speichern einer Datei:
    • Dateien speichern in: Wählen Sie diese Option, wenn &brandShortName; Dateien automatisch in den definierten Ordner speichern soll, ohne Sie nach zu fragen.
    • Immer nachfragen, wo Dateien gespeichert werden sollen: Wählen Sie diese Option, wenn Sie jedesmal wenn Sie eine Datei herunterladen gefragt werden wollen, wohin diese gespeichert werden soll. Standardmässig wird der Ordner vorgeschlagen, welcher das letzte Mal als Zielordner für einen Download gewählt wurde.
  • Download-Liste:
    • Download-Einträge entfernen: Diese Option bestimmt wann Einträge aus der Download-Manager Liste entfernt werden sollen (die heruntergeladenen Dateien selbst sind nicht betroffen):
      • Wenn Sie beendet sind: Wählen Sie diese Option wenn die Einträge sofort nach erfolgreichem Beenden des Downloads entfernt werden sollen.
      • Beim Beenden von &brandShortName;: Wählen Sie diese Option, wenn die Einträge erhalten bleiben sollen, bis Sie &brandShortName; beenden.
      • Nie: Wählen Sie diese Option, wenn die Einträge nie automatisch entfernt werden sollen. Sie können die Einträge händisch im Download-Manager entfernen.
  • Wenn ein Download abgeschlossen wird:
    • Klang abspielen: Wählen Sie diese Option wenn &brandShortName; einen selbst definierten Klang (üblicherwiese im WAV-Format) abspielen soll, sobald der Download beendet wird. Benutzen Sie den Durchsuchen-Button um einen Klang zu wählen. Um sich den Klang anzuhören, Klicken Sie auf Wiedergabe.
    • Warnung anzeigen: Wählen Sie diese Option wenn &brandShortName; eine Warnung anzeigen soll sobald ein Download abgeschlossen ist.

Für mehr Informationen über das Herunterladen von Dateien sehen Sie auch Plugins und Downloads.
