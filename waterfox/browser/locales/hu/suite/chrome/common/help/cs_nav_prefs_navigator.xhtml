Ez a fejezet a Böngésző beállításainak megadására szolgáló párbeszédpanelt ismerteti. A párbeszédpanel a következő lépésekkel hívható elő:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. Böngésző kiválasztása.
A fejezet témakörei:

Böngésző beállításai – Böngésző

Ez a témakör a Böngésző párbeszédpanel beállításait ismerteti. A párbeszédpanel a következő lépésekkel hívható elő:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. Kattintsunk a Böngésző kategóriára.

The Navigator preferences panel allows you to customize certain aspects of the browser.

  • Megjelenítés: Use the drop-down list in combination with the checkboxes to control what will be displayed at startup or when you open a new window or tab:
      • Select Navigator Startup to set what Navigator will display at startup.
      • Select New Window to set what will be displayed when opening a new Navigator window.
      • Select New Tab to set what will be displayed when opening a new tab.
    2. Ezen a párbeszédpanelen a böngésző három jellemzője állítható be:
      • Üres oldal: indításkor nem tölt be automatikusan weboldalt.
      • Kezdőlap: indításkor az alább megadott saját kezdőlap jelenik meg.
      • Az utoljára látogatott oldal: indításkor a &brandShortName; a program előző bezárásakor megjelenített oldalt nyitja meg .
      • Restore Previous Session: Causes the browser to restore the windows and tabs you were viewing right before you last exited &brandShortName;, including form data and browsing history. This option is only available in connection with Display on Browser Startup.
  • Alapértelmezett böngésző: itt beállítható a &brandShortName; alapértelmezett böngészőként, illetve ha már az, akkor az itt megjelenik.
    • Beállítás alapértelmezett böngészőként: ha a gomb nem szürke (inaktív), erre kattintva a &brandShortName; alapértelmezett böngészőként állítható be.
  • When restoring sessions and windows: Choose how many (if any) browser tabs should be restored at a time upon startup.
    • Restore all tabs immediately: All the tabs you had open the last time will be restored and loaded at once. This can affect startup performance negatively if a large amount of tabs needs to be restored in parallel.
    • Restore [__] tab(s) at a time: If you choose this, only the specified amount of tabs will be restored at a time. If you had more tabs open than specified here, any extra tabs will only be loaded once the loading of one of the other tabs has finished.
    • Only restore tabs when I need them: This disables automatic restoring of tabs, i.e. tabs will only be loaded if you switch to or reload them.
  • Kezdőlap: Írjuk be a mezőbe a kezdőlapnak szánt weboldal címét, vagy válasszunk a következő lehetőségek közül:
    • Aktuális oldal használata: ha ezt választjuk, a böngészőben jelenleg megjelenített oldal lesz a kezdőlap.
    • Aktuális csoport használata: ha egynél több böngészőlap van nyitva, ezzel a gombbal beállíthatjuk ezeket kezdőlapcsoportként (ez a lapcsoport nyílik meg kezdőlapként). A gomb megnyomása után a Kezdőlapcsoport van beállítva üzenet jelenik meg a hely mezőben.

      Vigyázat: ha az Aktuális csoport használata gomb megnyomása után szerkesztjük a mezőt, a kezdőlapcsoport elveszik.

    • Alaphelyzet: Click this to revert to the the default home page.
    • Tallózás: a gomb segítségével megkereshetjük a kezdőlapként beállítani kívánt oldalt a merevlemezen.
  • Válassza ki azokat a gombokat, amelyeket az eszköztáron szeretne látni: a megfelelő négyzetek bejelölésével megadhatjuk a kívánt gombokat.

    Az Ugrás, a Keresés és a Nyomtatás gomb a Böngésző eszköztár jobb felső sarkában, az összes többi a Személyes eszköztáron jelenik meg. Saját könyvjelzők felvételéről lásd: Személyes eszköztár.

Böngésző beállításai – Intelligens böngészés

Ez a témakör az Intelligens böngésző panellap beállításait ismerteti. If you're not already viewing it, follow these steps:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. Under the Navigator category, click Smart Browsing. (If no subcategories are visible, double-click Navigator to expand the list.)

The Smart Browsing preferences panel allows you to better browse web pages using Internet Keywords, Location Bar Autocomplete and Domain Guessing.

  • Internetes kulcsszavak:
    • Internetes kulcsszavak engedélyezése: Select this to enable fast access to services such as stock quotes, search, and other information from the Location bar.
    • More Information: Click this to learn more about using Internet Keywords.
  • Automatikus címkiegészítés:
    • Automatically complete text typed into Location bar: Select this to automatically complete text you previously entered into the Location bar.
    • Haladó: a gombra kattintva megadhatjuk a következő beállításokat is:
      • Atomatikus kiegészítés beírás közben: As you type in the Location Bar, &brandShortName; will automatically complete your web address using the visited website it most closely matches.
      • Lehetséges címek megjelenítése: As you type in the Location Bar, &brandShortName; will show a drop-down list of matching visited web addresses.
      • Internetes keresőszolgáltatás megjelenítése: Shows a drop-down list item allowing you to search the default search engine for words you enter.
      • Csak a már korábban beírt webhelyekre illeszkedik: Shows only websites that you've typed in the Location Bar and not sites that were opened in other ways, such as clicking a link on a web page.
  • Domain Guessing: Select this if you want &brandShortName; to automatically add www. and .com to a web page location that can't be found. For more detailed information about the Domain Guessing feature, see the online document Domain Guessing.

Böngésző beállításai – Böngészőlapok

Ez a témakör a Böngészőlapok panellap beállításait ismerteti. If you're not already viewing it, follow these steps:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. A Böngésző kategóriában kattintsunk a Böngészőlapok elemre. (Ha nem láthatóak a kategória beállításai, a lista kibontásához kattintsunk duplán a Böngésző kategóriára.)

Ezen a panellapon a Böngészőlapok beállításai adhatók meg:

  • Fülek megjelenítése:
    • A fül elrejtése, ha csak egy ablak látható: Select this to display the Tabbed Browsing bar only when more then one Navigator tab is open.
    • Új lap aktiválása a hivatkozás megnyitásakor: Select this to make &brandShortName; switch to the new tab when using Open in a New Tab to open a link.
    • Warn me when closing a window with multiple tabs: Select this to make &brandShortName; warn you when you try to close a browser window which has multiple tabs open in it.
    • Open related tabs after current tab: Select this to make new tabs open next to the tab from which they have been opened. When unchecked, new tabs open after the last tab on the tab bar.
  • Könyvjelzőcsoport megnyitásakor:
    • Új lapok hozzáadása: Select this if you want a bookmark group to be opened in new tabs.
    • Meglévő lapok lecserélése: Select this if you want a bookmark group to replace your existing tabs.
  • Megnyitás új ablak helyett, ha a felhasználó:
    • Cmd+click or Cmd+Return középső gombbal, Ctrl+kattintás vagy Ctrl+Enter a weboldal hivatkozásain : Select this to open Web page links in a new tab when clicking a link and holding down the Command key or holding down the Command key and pressing Return on links with the middle mouse button, Ctrl-clicking on links, and pressing Ctrl+Enter on links.

      Tip: Use Find Links as You Type to navigate to the link you want to open with the keyboard commands above.

    • CmdCtrl+ReturnEnter a címmezőben: Select this to open a Web page in a new tab when you type the URL of the page in the Location Bar and press CmdCtrl+ReturnEnter.
  • Hivatkozás megnyitása: If a webpage is designed so that certain links open in a new window by default (either through the target attribute in HTML or through embedded JavaScript), you may want to override this:
    • Az aktuális lapon/ablakban: Open the linked page in the current tab of the active window.
    • Új lapon az aktuális ablakban: Open the linked page in a new tab instead of a new window.
    • Új ablakban: Open the linked page in a new window. (This is the default and does not override the webpage design).
  • Más alkalmazásokból megnyitott hivatkozások: If &brandShortName; is called from another application with a webpage address as an argument (like a click on a link in an external email program), you can control where the page will be loaded:
    • Az aktuális lapon/ablakban: Open the linked page in the current tab of the active window.
    • Új lapon az aktuális ablakban: Open the linked page in a new tab instead of a new window.
    • Új ablakban: Open the linked page in a new window. (This is the default).

To ensure that &brandShortName; opens a new window, select the Navigator option in Appearance Preferences - Appearance.

If you want &brandShortName; to open a new tab instead of a new window when you launch it and it is already running, ensure the following:

  • If you launch &brandShortName; by command line, do not use command-line parameters that open windows.
  • Nothing is set in the Megjelenés beállításai - Megjelenés.
  • Set Links from other applications preference to A new tab in the current window.

Böngésző beállításai – Letöltések

Ez a témakör a Letöltések panellap beállításait ismerteti. If you're not already viewing it, follow these steps:

  1. Válasszuk a &brandShortName; Szerkesztés menüjéből a Beállítások parancsot.
  2. A Böngésző kategóriában kattintsunk a Letöltések elemre. (Ha nem láthatóak a kategória beállításai, a lista kibontásához kattintsunk duplán a Böngésző kategóriára.)

The Downloads preferences panel allows you to set up how &brandShortName; handles files you download from web pages:

  • A letöltés kezdetekor:
    • Letöltéskezelő megnyitása: választásakor megnyílik a Letöltéskezelő, amely az aktuális és a korábbi letöltések állapotát jeleníti meg. Az összes letöltés egyetlen ablakban kezelhető.
    • Folyamatkijelző ablak megnyitása: Select this to display a progress dialog box, which display the status for your current download. Az egyes letöltésekhez külön párbeszédpanel tartozik.
    • Ne nyisson meg semmit: Select this if you want to download files invisibly. No status is given for all your downloads.
  • Fájl mentésekor:
    • Letöltési hely megkérdezése és alapértelmezett hely: Select this if you always want to be able to choose a folder for the file to be saved. Choose one of the following folders to be the default location for the file:
      • Utoljára használt mappa: The default location will be the folder you last downloaded a file to.
      • Megadott letöltési mappa: The default location will be your Current Download Folder.
    • Fájlok automatikus letöltése a megadott letöltési mappába: Select this if you want files to be saved to your Current Download Folder without &brandShortName; prompting you for the download location.
  • Aktuális letöltési mappa: Use the button to navigate to a folder that you want to use as your specified download folder.
  • Letöltés befejezésekor:
    • Hangjelzés: Select this if you want &brandShortName; to play a custom sound (typically in WAV format) when a download is completed. Use the Browse button to select the sound file in the file locator. To listen to the sound you've chosen, click Preview.
    • Figyelmeztető ablak: Select this if you want &brandShortName; to show an alert on the screen when a download is completed.

For more information about downloading files from web pages, see Plug-ins and Downloads.
