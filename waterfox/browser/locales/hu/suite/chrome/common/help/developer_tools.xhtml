Webfejlesztést segítő eszközök

A &brandShortName; több eszközt is nyújt a webfejlesztők számára a webalkalmazások fejlesztéséhez és hibakereséséhez. Ezek egy része választható összetevő a telepítéskor. Ezeket az eszközöket a Eszközök menü Webfejlesztés almenüjében találhatjuk meg.

  • Hibanaplózó konzol: a konzolablakban a JavaScript és a CSS kód figyelmeztető és hibaüzenetei jelennek meg, akár a webalkalmazásban, akár magában a &brandShortName; alkalmazás okozta őket. By default, CSS parsing errors and JavaScript errors are displayed. The console can also be used to display logged messages from XUL and JavaScript code.
  • DOM Inspector: eszköz, amellyel tetszőleges webdokumentum vagy XUL-alkalmazás DOM-ját lehet vizsgálni és szerkeszteni.

    Online források:

    • Introduction to the DOM Inspector (Bevezetés a DOM Inspector használatába) (Ian Oeschger)
    • DOM Inspector FAQ (Gyakran feltett kérdések) (mozilla.org)
    • Creating Applications with Waterfox - Appendix B3 (Alkalmazások készítése a Mozillával – B3 függelék): The DOM Inspector
  • JavaScript hibakereső: másnéven Venkman. A segédprogram lehetővé teszi a hibakeresést JavaSript programokban a &brandShortName; alatt. Támogatja a lépésenkénti végrehajtást, a töréspontokat és még sok más egyéb funkciója van.
